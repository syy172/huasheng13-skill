## 效果示例

> **场景一：题目讲解**

```
用户      ❯ 缔约过失责任是指在合同订立过程中，一方因违背依据诚实信用原则所产生的义务，而致另一方的信赖利益受损失，就应承担损害赔偿责任。
根据以上定义，下列选项中乙方不可以要求甲方负缔约过失责任的是（　　）。
A
甲、乙公司经过多轮磋商对机床价格达成了一致，于是乙公司派人来与甲公司签约，甲公司突然加价，乙公司未同意，甲公司遂拒绝签约，乙公司因此不能及时开工
B
甲公司得知乙公司正在就贸易合同与丙公司谈判，为了排挤丙公司，就向乙公司提出了更优惠的条件。丙公司退出后，甲公司借故终止谈判，乙公司因此受损
C
甲厂与乙厂达成了药材供应协议，约定每月向乙厂供药材10吨，后药材价格暴涨，甲厂希望能重新商定价格，遂停止供货，乙厂因此失去大批客户
D
甲公司与乙公司就彩电购销协议进行洽谈，洽谈过程中甲公司接触到了乙公司保密的市场开发计划，遂拖延洽谈过程，暗中吸引乙的目标客户，致乙市场份额锐减

公考.skill ❯ ## 答案：C

**核心区分点：合同是否已成立。**

缔约过失责任发生在**合同订立过程中**，即合同**尚未成立**。若合同已经成立，则一方不履行义务构成的是**违约责任**，而非缔约过失责任。

逐项分析：

| 选项 | 合同状态 | 情形 | 结论 |
|------|---------|------|------|
| **A** | 未成立 | 磋商签约阶段，甲突然加价拒绝签约 | 可主张缔约过失 ✓ |
| **B** | 未成立 | 甲假借磋商恶意排挤丙，然后借故终止 | 典型恶意磋商，缔约过失 ✓ |
| **C** | **已成立** | 甲乙**已达成协议**，甲因涨价擅自停止供货 | **违约责任**，非缔约过失 ✗ |
| **D** | 未成立 | 洽谈中甲窃取乙保密商业信息并恶意利用 | 违反先合同义务，缔约过失 ✓ |

**C是正确答案。** 题目明确写了"达成了药材供应协议，约定每月向乙厂供药材10吨"——协议已成立并生效，甲单方停止供货属于**违约**，乙应主张违约责任，而非缔约过失责任。
```

> **场景二：资料分析提速**

```
用户      ❯ 资料分析怎么提速到25分钟20题？

公考.skill ❯ 速度瓶颈诊断：

             ① 读材料太慢 → 先看题再定位，别通读
             ② 列式犹豫 → 记熟ABRX四要素，看到"增长"就反应
             ③ 计算太慢 → 截位直除保留前3位，别死算
             ④ 纠结选项 → 差距大直接选，差距小才精算
             ⑤ 时间分配不均 → 每篇6-7分钟，超时就跳

             三大速算速成：
             ├─ 截位直除：分子分母各取前3位，一步出结果
             ├─ 415份数：看到"增长25%"→ 原来4份现在5份
             └─ 假设分配：多步增长率，假设基期反推

             训练计划：每日2篇 → 4篇 → 套题，2周见效。
```

> **场景三：数量关系蒙题**

```
用户      ❯ 数量关系全蒙C行不行？

公考.skill ❯ 全蒙C期望得分：15题×0.2概率×1分 = 3分。亏。

             选做策略（5-8分钟挑5道会的）：
             ├─ 工程问题：赋特殊值，设总量为公倍数
             ├─ 利润问题：成本100，利润率直接算
             ├─ 容斥问题：两集合A+B-A∩B=总数
             ├─ 年龄问题：年龄差不变，列表代入
             └─ 几何问题：特殊图形代入，比例缩放

             剩余10题蒙题技巧：
             ① 排除极值（最大最小一般不选）
             ② 选项分布（一般ABCD各3-4个）
             ③ 看到"约"选最近的整数

```

> **场景四：申论大作文**

```
用户      ❯ 申论大作文怎么写？

公考.skill ❯ 审题三步：
             ① 题干关键词 → 确定主题方向
             ② 材料主旨 → 找核心论点
             ③ 要求 → 字数、格式、身份

             结构推荐（五段三分式）：
             ┌─ 开头：引出问题 + 亮明观点（150字）
             ├─ 分论点1：是什么 + 为什么 + 怎么办（250字）

             ├─ 分论点3：同上（250字）

             论证方法：例证（案例）+ 理证（道理）+ 对比
             开头模板：引言式/案例式/排比式/设问式
```

## 答题策略

### 推荐顺序（花生十三版）

```
常识（8min）→ 资料分析（28min）→ 判断推理（35min）→ 言语理解（35min）→ 数量关系（剩余时间选做）
```

### 时间分配原则

1. **先易后难、先高分后低分**：资料分析分值密度最高，必须先做且做对
2. **数量关系放最后**：性价比最低，留5-8分钟挑5-7道会做的，其余统一蒙
3. **常识不纠结**：会就选，不会就蒙，单题不超过30秒
4. **涂卡分两次**：做完资料涂一次，做完言语判断涂一次，避免最后来不及

### 涂卡时机

```
做完常识 → 涂卡
做完资料分析 → 涂卡
做完判断推理 → 涂卡
做完言语理解 → 涂卡
做完数量关系 → 涂卡
```

## 功能特性

### 覆盖题型

**资料分析**：基期/现期、增长量/率、比重、平均数、倍数、综合分析

**数量关系**：工程、行程、利润、排列组合、概率、容斥、最值、年龄、几何、溶液、牛吃草、鸡兔同笼

**言语理解**：逻辑填空（语境分析、词义辨析）、片段阅读（中心理解、细节判断、标题填入、语句填空、下文推断）、语句排序、篇章阅读

**判断推理**：图形推理（位置/样式/数量/属性规律）、定义判断（要素法）、类比推理（语义/逻辑/语法关系）、逻辑判断（翻译推理、真假推理、加强削弱、日常结论、解释评价）

**申论**：单一题（归纳概括）、综合分析题、提出对策题、公文写作题、大作文（议论文）

### 输出规范

每道题的讲解包含：

- **题型识别**：关键词 → 判断题型
- **解题思路**：识别标志 → 套公式/套方法 → 速算 → 选答案
- **计算过程**：完整的速算步骤
- **易错点**：常见错误提醒
- **同类拓展**：相关题型变式

## 注意事项

* **速算必须给过程**：资料分析、数量关系的题，不能只给答案，必须展示速算过程
* **方法优先于计算**：先讲识别标志和解题思路，再讲计算
* **易错点必给**：每道题、每个方法都要附易错点
* **真题示例用通用数据**：使用公开真题或自编数据，不照搬付费课程原题
* **申论大作文给框架**：不给完整范文，给"立意 + 结构 + 分论点 + 素材方向"
* **备考建议因人而异**：必须先问清用户水平、时间、目标，再给计划
