# 公考花生十三 Skill

基于花生十三公开教学体系整理的公务员考试（国考/省考/联考/事业单位）行测+申论解题技巧库。

## 📚 项目简介

本技能是一个全面的公务员考试备考助手，覆盖六大模块、二十余题型、上百条速算与解题公式，配真题示例与实战复盘方法。基于花生十三老师的公开教学体系整理，适用于国考、省考、联考及事业单位考试。

## ✨ 功能特点

- **全面覆盖**：资料分析、数量关系、言语理解、判断推理、常识判断、申论六大模块
- **速算技巧**：截位直除、415份数法、假设分配法、ABRX体系等高效解题方法
- **真题实战**：提供各模块真题示例与详细解析
- **备考规划**：个性化备考计划、答题顺序、时间分配策略
- **错题复盘**：错题归因分析、复盘模板、薄弱环节诊断

## ⏱️ 答题顺序与时间分配

### 推荐顺序（花生十三公开建议版）

```
常识（8 min）→ 资料分析（28 min）→ 判断推理（35 min）→ 言语理解（35 min）→ 数量关系（剩余时间选做）
```

### 时间分配原则

1. **先易后难、先高分后低分**：资料分析分值密度最高，必须先做且做对
2. **数量关系放最后**：性价比最低，留5–8分钟挑5–7道会做的，其余统一蒙
3. **常识不纠结**：会就选，不会就蒙，单题不超过30秒
4. **涂卡分两次**：做完资料涂一次，做完言语判断涂一次，避免最后来不及

  1. 定位材料：找到现期量与基期量
  2. 套公式：增长率 =（现期 - 基期）/ 基期

【答案】X
【易错点】① 时间定位错 ② 单位（万/亿）③ 增长量与增长率混淆

## 🔧 使用约束

1. **速算必须给过程**：资料分析、数量关系的题，不能只给答案，必须展示速算过程
2. **方法优先于计算**：先讲识别标志和解题思路，再讲计算
3. **易错点必给**：每道题、每个方法都要附易错点
4. **真题示例用通用数据**：使用公开真题或自编数据，不照搬付费课程原题
5. **申论大作文给框架**：不给完整范文，给"立意 + 结构 + 分论点 + 素材方向"
6. **备考建议因人而异**：必须先问清用户水平、时间、目标，再给计划
