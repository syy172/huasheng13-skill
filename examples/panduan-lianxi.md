## 一、图形推理

### 练习 1：位置规律（平移）

**题目**：给出 4 个图形，黑点位置依次为：左上、右上、右下、左下。第 5 个图形黑点应在？

**解析**：
- 元素相同 → 位置规律
- 黑点顺时针移动
- 第 5 个 = 左上

**答案**：左上

---

### 练习 2：位置规律（旋转）

**题目**：给出 4 个图形，依次顺时针旋转 90°。第 5 个图形是？

**解析**：
- 元素相同 → 位置规律
- 每次顺时针旋转 90°
- 第 5 个 = 第 4 个顺时针转 90°

### 练习 3：样式规律（黑白叠加）

**题目**：图 1 + 图 2 = 图 3，规则为黑+黑=白，白+白=白，黑+白=黑。求图 4 + 图 5 = ?

**解析**：
- 元素相似 → 样式规律
- 应用叠加规则
- 逐个位置叠加

---

### 练习 4：数量规律（封闭区域）

**题目**：给出 5 个图形，封闭区域数依次为 3, 4, 5, 6, 7。第 6 个图形应有几个封闭区域？

**解析**：
- 元素不同 → 数量规律
- 数封闭区域：3, 4, 5, 6, 7 → 等差数列
- 第 6 个 = 8

**答案**：8 个封闭区域

### 练习 6：属性规律（对称）

## 二、定义判断

### 练习 7：单定义

**定义**：锚定效应是指人们在做决策时，过度依赖最初获得的信息（锚），并以此为基准调整后续判断的现象。

**问题**：以下属于锚定效应的是：
A. 小明看到商品原价 999 元，现价 499 元，觉得很划算
B. 小红比较了多家店铺后选择最便宜的
C. 小李根据朋友推荐购买商品
D. 小张凭直觉做决定

**解析**：
- 要素：①做决策时 ②依赖最初信息（锚）③以此为基准调整
- A：看到原价 999（锚）→ 觉得 499 划算 → 符合
- B：比较多家，无"最初信息"锚定 → 不符合
- C：朋友推荐，非"最初信息"锚定 → 不符合
- D：凭直觉，无锚 → 不符合

**答案**：A

### 练习 8：单定义（多要素）

**定义**：破窗效应是指如果一幢建筑物的窗户玻璃坏了又得不到及时的维修，别人就可能受到暗示性纵容去打烂更多的窗户玻璃。

**问题**：以下属于破窗效应的是：
A. 小明看到地上有垃圾，也扔了垃圾
B. 小红看到别人闯红灯，也跟着闯
C. 小李看到墙上有人涂鸦，也去涂鸦
D. 以上都是

**解析**：
- 要素：①不良现象未及时制止 ②暗示他人效仿
- A、B、C 都符合
- 选 D

**答案**：D

## 四、逻辑判断

### 练习 13：翻译推理

**题目**：如果天下雨，那么地会湿。以下哪项一定正确？
A. 地湿了，说明天下雨了
B. 天没下雨，地一定没湿
C. 地没湿，说明天没下雨
D. 天下雨了，地一定湿

**解析**：
- 翻译：下雨 → 地湿
- A 肯后，错
- B 否前，错
- C 逆否（¬地湿 → ¬下雨），对
- D 肯前，对

**答案**：D（"一定正确"，D 是直接推理）

---

### 练习 14：翻译推理（且或）

**题目**：甲说："如果我去，那么乙或丙也去。"已知甲去了，则：
A. 乙去了
B. 丙去了
C. 乙和丙都去了
D. 乙和丙至少一人去了

**解析**：
- 翻译：甲去 → 乙去 ∨ 丙去
- 甲去了（肯前）→ 乙去 ∨ 丙去
- 即乙和丙至少一人去

**答案**：D

### 练习 15：加强题

**论证**：研究发现，每天喝咖啡的人患心脏病的概率低于不喝咖啡的人。因此，喝咖啡能预防心脏病。

**问题**：以下哪项最能加强？
A. 咖啡含有抗氧化物质
B. 喝咖啡的人通常也更注重锻炼
C. 有些不喝咖啡的人也患心脏病
D. 咖啡的价格较高

**解析**：
- 论点：喝咖啡能预防心脏病
- 论据：喝咖啡的人患病率低
- 加强方法：建立联系 / 解释原因
- A：咖啡含抗氧化物质，解释了为什么能预防 → 加强
- B：另有他因，削弱
- C：举例，无关
- D：无关

**答案**：A

---

### 练习 16：削弱题

**论证**：研究发现，每天喝咖啡的人患心脏病的概率低于不喝咖啡的人。因此，喝咖啡能预防心脏病。

**问题**：以下哪项最能削弱？
A. 咖啡含有咖啡因，可能影响睡眠
B. 每天喝咖啡的人通常也更注重锻炼和饮食
C. 有些喝咖啡的人也患心脏病
D. 咖啡的价格较高

**解析**：
- 论点：喝咖啡能预防心脏病
- 论据：喝咖啡的人患病率低
- 削弱方法：另有他因
- B：指出喝咖啡的人本身更健康（锻炼饮食），是这些因素导致患病率低 → 另有他因，削弱
- A、D 无关
- C 举例反驳，力度弱

**答案**：B

### 练习 17：真假推理

**题目**：甲乙丙三人，一人说真话，两人说假话。
甲说："乙说假话。"
乙说："丙说假话。"
丙说："甲和乙都说假话。"

谁说真话？

**解析**：
- 找矛盾：甲说"乙假"，乙说"丙假"，丙说"甲乙都假"
- 假设甲真：则乙假，乙假则丙真，丙真则甲乙都假，矛盾（甲真）
- 假设乙真：则丙假，丙假则甲乙不都假，即至少一人真，乙真符合，甲呢？甲说"乙假"是假话，所以甲假，符合"不都假"
- 乙真，甲假，丙假

**答案**：乙说真话

### 练习 18：朴素逻辑

**题目**：A、B、C 三人，分别是教师、医生、律师（顺序不一定对应）。
① A 不是教师
② B 不是医生
③ C 不是律师
④ 教师比 C 年龄大
⑤ B 比医生年龄大

求各人职业。

**解析**：
- 由④：C 不是教师（教师比 C 大）
- 由①：A 不是教师
- 所以 B 是教师
- 由②：B 不是医生（已知 B 是教师，符合）
- 由⑤：B（教师）比医生大
- 由④：教师（B）比 C 大
- 所以 C 不是医生（因为 B > 医生，B > C，无法确定 C 和医生关系）
- 等等，重新分析：
- B 是教师
- 由⑤：B > 医生，即医生比 B 小
- 由④：B > C
- C 和医生都比 B 小，但 C 不是医生（因为如果 C 是医生，则 B > C 满足，但还要看其他条件）
- 由③：C 不是律师
- C 不是教师（B 是），不是律师，所以 C 是医生
- 但 C 是医生，则 B > C（医生），满足⑤
- 剩下 A：不是教师（B 是），不是医生（C 是），所以 A 是律师
- 验证③：C 不是律师 ✓

**答案**：A 是律师，B 是教师，C 是医生

## 配套速记

- **图形**：相同位置，相似样式，不同数量
- **定义**：抓要素，逐一比对
- **类比**：看关系，辨方向
- **翻译**：前推后，肯前否后
- **加强**：建立联系最强
- **削弱**：另有他因中等
- **真假**：找矛盾，绕开看
- **朴素**：列表排除
