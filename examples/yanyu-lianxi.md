## 一、逻辑填空

### 练习 1：转折对应

**题目**：这项工作看似______，实则______，需要我们认真对待。
A. 简单 / 复杂  B. 容易 / 困难  C. 简单 / 简单  D. 复杂 / 简单

**解析**：
- "看似...实则..."表转折，前后反义
- A：简单 vs 复杂，反义 ✓
- B：容易 vs 困难，反义 ✓
- C：简单 vs 简单，不反义 ✗
- D：复杂 vs 简单，反义，但逻辑反了（看似复杂实则简单，与"认真对待"不符）
- A、B 都符合，二级辨析："简单/复杂"更书面

**答案**：A

---

### 练习 2：并列对应

**题目**：他不仅聪明，而且______。
A. 勤奋  B. 懒惰  C. 愚蠢  D. 平凡

**解析**：
- "不仅...而且..."表递进，前后近义或递进
- A 勤奋：与聪明并列，积极 ✓
- B 懒惰：与聪明反义 ✗
- C 愚蠢：与聪明反义 ✗
- D 平凡：与聪明不并列 ✗

**答案**：A

---

### 练习 3：解释对应

**题目**：他是个______的人，即不轻易发表意见。
A. 沉默  B. 活泼  C. 开朗  D. 健谈

**解析**：
- "即"表解释，后文解释前文
- "不轻易发表意见"→ 沉默

**答案**：A

---

### 练习 4：成语辨析（程度）

**题目**：面对这种______的行为，我们必须严厉打击。
A. 不良  B. 恶劣  C. 错误  D. 不当

**解析**：
- "严厉打击"→ 程度重
- A 不良：程度轻
- B 恶劣：程度重 ✓
- C 错误：程度轻
- D 不当：程度轻

**答案**：B

---

### 练习 5：成语辨析（感情色彩）

**题目**：他______地完成了任务，受到领导表扬。
A. 处心积虑  B. 煞费苦心  C. 殚精竭虑  D. 苦心孤诣

**解析**：
- "受到表扬"→ 褒义
- A 处心积虑：贬义 ✗
- B 煞费苦心：中性偏褒
- C 殚精竭虑：褒义 ✓
- D 苦心孤诣：褒义
- C 最符合"完成任务受表扬"

**答案**：C

### 练习 6：两空题

**题目**：传统文化需要______，在传承中创新，否则就会______。
A. 停滞不前 / 故步自封  B. 故步自封 / 日渐式微  C. 日渐式微 / 墨守成规  D. 薪火相传 / 日渐式微

**解析**：
- 第一空：传承文化，需要怎样
  - A 停滞不前：不前进 ✗
  - B 故步自封：自满 ✗
  - C 日渐式微：衰落 ✗
  - D 薪火相传：传承 ✓
- 第二空：不创新就会怎样
  - D 日渐式微：衰落 ✓
- 选 D

**答案**：D

## 二、片段阅读

### 练习 7：中心理解（转折）

**题目**：随着人工智能技术的发展，越来越多的工作岗位被机器取代。然而，技术进步也创造了大量新的就业机会，如数据标注师、AI 训练师等新兴职业不断涌现。因此，面对人工智能，我们不必过度恐慌，而应主动适应，提升技能。

这段文字意在说明：
A. 人工智能会取代大量工作岗位
B. 人工智能创造了新的就业机会
C. 应对人工智能应主动适应、提升技能
D. 人工智能技术发展迅速

**解析**：
- 关联词："然而"（转折）→ 重点在后
- "因此"（因果）→ 结论是重点
- 结论："不必恐慌，应主动适应，提升技能"
- 对应 C

**答案**：C

### 练习 8：中心理解（递进）

**题目**：读书可以增长知识，可以开阔视野，更可以塑造人格。一本好书，往往能影响人的一生。

这段文字主要说明：
A. 读书可以增长知识
B. 读书可以开阔视野
C. 读书可以塑造人格
D. 读书意义重大

**解析**：
- "更"表递进，重点在递进后
- 但三个分句都是读书的意义
- 全面概括 → D

**答案**：D

### 练习 9：细节判断

**题目**：根据文段，以下说法正确的是：
（文段：人工智能发展迅速，创造了新的就业机会，但也取代了一些工作岗位。）

A. 人工智能将完全取代人类工作
B. 人工智能创造了新的就业机会
C. 所有人都应该学习人工智能技术
D. 人工智能技术发展已经成熟

**解析**：
- A "完全取代"过于绝对，文段说"一些"不是"完全" ✗
- B 文段提到"创造了新的就业机会" ✓
- C "所有人"过于绝对 ✗
- D "已经成熟"文段未提 ✗

**答案**：B

### 练习 10：标题填入

**题目**：随着城市化进程加快，越来越多的农村人口涌入城市。这既为城市发展提供了劳动力，也带来了住房、教育、医疗等压力。如何平衡城市化与城市承载能力，成为亟待解决的问题。

最适合的标题是：
A. 城市化的利与弊
B. 农村人口涌入城市
C. 城市化进程加快
D. 平衡城市化与承载能力

**解析**：
- 文段最后一句是中心："如何平衡...成为亟待解决的问题"
- 标题应反映主旨
- D 最准确

**答案**：D

### 练习 11：语句衔接

**题目**：______。因此，我们要重视环境保护，走可持续发展之路。
A. 环境污染日益严重
B. 经济发展是硬道理
C. 科技发展日新月异
D. 文化传承任重道远

**解析**：
- "因此"表因果，前文是原因
- 后文结论："重视环保，可持续发展"
- 原因应是环境问题
- A 环境污染严重 ✓

**答案**：A

### 练习 12：语句排序

**题目**：① 这种现象引起了社会的广泛关注
② 近年来，共享单车乱停乱放问题日益严重
③ 一些城市开始出台管理办法
④ 同时，企业也加强了运营管理
⑤ 共享单车行业逐渐走向规范

A. ②①③④⑤  B. ②①④③⑤  C. ①②③④⑤  D. ②③①④⑤

**解析**：
- 首句：①"这种现象"含代词，不能做首句 → 排除 C
- ②引出话题"共享单车乱停乱放"，适合做首句
- ①"这种现象"指代②的现象 → ②①捆绑
- ③"出台管理办法"和④"企业加强管理"是并列应对
- ⑤"走向规范"是结果，做尾句
- 顺序：②①③④⑤

**答案**：A

## 三、语句表达

### 练习 13：语句排序

**题目**：① 它是中华民族的精神命脉
② 传统文化是民族的根
③ 没有它，就没有中华民族
④ 也是我们文化自信的源泉

A. ②①④③  B. ②①③④  C. ①②③④  D. ②④①③

**解析**：
- 首句：①"它"含代词，不做首句 → 排除 C
- ②引出"传统文化"，适合做首句
- ①"它"指代②的"传统文化" → ②①捆绑
- ④"也是"承接①"是"
- ③"没有它"总结
- 顺序：②①④③

**答案**：A

## 配套速记

- **填空**：找对应（转折反义、并列近义、递进加深、解释同义）
- **成语**：辨程度、辨色彩、辨对象
- **中心**：关联词定重点，主题词必含
- **细节**：防六种偷换，绝对化多为错
- **标题**：反映主旨，简洁吸引
- **衔接**：看位置，找话题
- **排序**：代词不做首，捆绑定顺序
- **篇章**：先看题后读文，定位找答案
