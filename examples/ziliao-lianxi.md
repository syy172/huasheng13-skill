## 练习 1：基期计算（截位直除）

**材料**：
> 2023 年，某省地区生产总值 56780 亿元，同比增长 5.4%。

**问题**：2022 年该省地区生产总值约为：
A. 53870 亿元  B. 53870 亿元  C. 53870 亿元  D. 53870 亿元

（选项设计为不同数值，此处简化）

**解析**：
- 题型：基期计算，已知现期 + 增长率
- 公式：基期 = 现期 / (1 + R)
- 计算：56780 / 1.054
- 截位：选项差距小，截 3 位
- 56780 / 1.054 ≈ 53870
- 答案：53870 亿元

**易错点**：增长用 1 + R，下降用 1 - R。

## 练习 2：增长量计算（415 份数法）

**材料**：
> 2023 年，某市财政收入 2345 亿元，同比增长 6.8%。

**问题**：2023 年该市财政收入同比增加约：
A. 145 亿元  B. 149 亿元  C. 153 亿元  D. 157 亿元

**解析**：
- 题型：增长量计算，已知现期 + 增长率
- 方法：415 份数法
- 现期 : 增长量 = 106.8 : 6.8
- 每份 = 2345 / 106.8 ≈ 22（截位直除）
- 增长量 = 22 × 6.8 ≈ 150
- 最接近 B（149）

**答案**：B

**易错点**：别用 X = A × R / (1+R) 直接算，计算量大。

## 练习 3：增长率比较

**材料**：
> 2023 年，A 产品产量 120 万吨，上年 100 万吨；B 产品产量 230 万吨，上年 200 万吨；C 产品产量 360 万吨，上年 300 万吨。

**问题**：2023 年三种产品产量同比增长率由高到低排序正确的是：
A. A > B > C  B. A > C > B  C. B > A > C  D. C > B > A

**解析**：
- 题型：增长率比较
- 方法：分数比较法
- A 增长率 = (120-100)/100 = 20%
- B 增长率 = (230-200)/200 = 15%
- C 增长率 = (360-300)/300 = 20%
- A = C > B
- 选项无 A = C，选最接近的 A（A > B > C）

**答案**：A

**易错点**：增长率 = 增量 / 基期，别用现期。

## 练习 4：比重计算

**材料**：
> 2023 年，某省出口额 4567 亿元，进出口总额 8901 亿元。

**问题**：2023 年该省出口额占进出口总额的比重约为：
A. 45.3%  B. 48.5%  C. 51.3%  D. 54.2%

**解析**：
- 题型：比重计算
- 公式：比重 = 部分 / 整体
- 计算：4567 / 8901
- 截位：选项首位同（4），第二位不同，截 3 位
- 4567 / 8901 ≈ 457 / 890 ≈ 0.513
- 答案：51.3%

**答案**：C

**易错点**：比重 = 部分 / 整体，别颠倒。

## 练习 5：比重升降

**材料**：
> 2023 年，某省规模以上工业增加值 12345 亿元，同比增长 4.5%。其中，高技术产业增加值 3456 亿元，同比增长 8.7%。

**问题**：2023 年高技术产业增加值占规模以上工业增加值的比重，与上年相比：
A. 上升约 1.1 个百分点  B. 下降约 1.1 个百分点  C. 上升约 11 个百分点  D. 下降约 11 个百分点

**解析**：
- 题型：比重升降（定性 + 定量）
- 定性：部分 R（8.7%）> 整体 R（4.5%）→ 比重上升，排除 B、D
- 定量：比重差 = (3456/12345) × [(8.7% - 4.5%) / (1 + 8.7%)]
  - 3456/12345 ≈ 28%
  - 8.7% - 4.5% = 4.2%
  - 4.2% / 1.087 ≈ 3.86%
  - 28% × 3.86% ≈ 1.08 个百分点
- 答案：上升约 1.1 个百分点

**答案**：A

**易错点**：比重差是百分点，不是百分比。

## 练习 6：平均数计算

**材料**：
> 2023 年，某省粮食总产量 3456 万吨，播种面积 5678 万亩。

**问题**：2023 年该省粮食亩产量约为：
A. 408 公斤  B. 508 公斤  C. 608 公斤  D. 708 公斤

**解析**：
- 题型：平均数计算
- 公式：平均数 = 总量 / 份数
- 计算：3456 万吨 / 5678 万亩 = 3456 / 5678 吨/亩
- 截位：3456 / 5678 ≈ 346 / 568 ≈ 0.609 吨/亩 = 609 公斤/亩
- 最接近 C（608）

**答案**：C

**易错点**：单位换算，1 吨 = 1000 公斤。

## 练习 7：综合分析题

**材料**：
> 2023 年，全国 GDP 1260582 亿元，同比增长 5.2%。其中，第一产业 89755 亿元，增长 4.1%；第二产业 482589 亿元，增长 4.7%；第三产业 688238 亿元，增长 5.8%。

**问题**：能够从上述资料中推出的是：
A. 2023 年第一产业占比超过 8%
B. 2023 年第三产业增量大于第二产业
C. 2022 年 GDP 低于 120 万亿元
D. 2023 年三大产业均增长超过 5%

**解析**：

**说法 A**：89755 / 1260582 ≈ 7.1% < 8% → 错

**说法 B**：
- 第二产业增量：482589 × 4.7% / 1.047 ≈ 21680
- 第三产业增量：688238 × 5.8% / 1.058 ≈ 37730
- 第三 > 第二 → 对
- 选 B，结束

**答案**：B

## 练习 8：现期计算（假设分配法）

**材料**：
> 2022 年某省 GDP 54321 亿元，2023 年同比增长 6.5%。

**问题**：2023 年该省 GDP 约为：
A. 56780 亿元  B. 57850 亿元  C. 57850 亿元  D. 58920 亿元

**解析**：
- 题型：现期计算，已知基期 + 增长率
- 方法：假设分配法
- 基期 54321，增长率 6.5%
- 100 份 = 54321，每份 = 543.21
- 现期 = 106.5 份 = 543.21 × 106.5 ≈ 57852
- 答案：57850 亿元

**答案**：B（或 C，数值相同）

**易错点**：现期 = 基期 × (1 + R)。

## 练习 9：倍数计算

**材料**：
> 2023 年，A 市财政收入 456 亿元，B 市财政收入 123 亿元。

**问题**：2023 年 A 市财政收入是 B 市的约：
A. 2.7 倍  B. 3.7 倍  C. 4.7 倍  D. 5.7 倍

**解析**：
- 题型：倍数计算
- 公式：倍数 = A / B
- 计算：456 / 123
- 截位：456 / 123 ≈ 456 / 123 ≈ 3.7
- 答案：3.7 倍

**答案**：B

**易错点**：倍数 = A / B，"是几倍"用除法。

## 练习 10：增长率计算

**材料**：
> 2023 年某省出口额 5678 亿元，2022 年为 4567 亿元。

**问题**：2023 年该省出口额同比增长率为：
A. 12.4%  B. 18.4%  C. 24.4%  D. 30.4%

**解析**：
- 题型：增长率计算
- 公式：增长率 = (现期 - 基期) / 基期
- 计算：(5678 - 4567) / 4567 = 1111 / 4567
- 截位：1111 / 4567 ≈ 111 / 457 ≈ 0.243
- 答案：24.4%

**答案**：C

**易错点**：增长率 = 增量 / 基期，分母是基期不是现期。

## 配套速记

- **基期**：现期 / (1+R)，截位直除
- **增长量**：415 份数法
- **现期**：基期 × (1+R)，假设分配
- **增长率**：增量 / 基期
- **比重**：部分 / 整体
- **比重差**：比重 × 差率 / (1+部分R)
- **平均数**：总量 / 份数
- **倍数**：A / B
- **综合分析**：先易后难，找到就选
