# 资料分析（花生十三体系）

资料分析是行测分值密度最高、提分最快的模块。花生十三体系的核心是 **ABRX 四要素 + 三大速算方法（截位直除、415 份数、假设分配）**，掌握后可实现 25 分钟内完成 20 题、正确率 90% 以上。

---

## 一、ABRX 四要素体系

资料分析所有题目本质都是在算四个量之间的关系：

| 符号 | 含义 | 别名 | 公式 |
|------|------|------|------|
| **A** | 现期量 | 今年 / 本期 / 报告期 | A = B × (1 + X) |
| **B** | 基期量 | 去年 / 上期 / 基期 | B = A / (1 + X) |
| **R** | 增长率 | 增速 / 幅度 / 百分比 | R = (A - B) / B = X / B |
| **X** | 增长量 | 净增额 / 增量 | X = A - B = B × R |

**核心记忆**：知任意两个量，可求另外两个。资料分析 90% 的题都是"已知两个求第三个"。

### 四要素互推关系图

```
        A（现期）
       / \
      /   \
   1+R     A/(1+R)
    /         \
   /           \
  R（增长率）—— B（基期）
       |
     B×R
       |
       X（增长量）
```

### 常见题型与对应公式

| 题型 | 已知 | 求 | 公式 |
|------|------|---|------|
| 基期计算 | A、R | B | B = A / (1 + R) |
| 基期计算 | A、X | B | B = A - X |
| 增长量计算 | A、R | X | X = A × R / (1 + R) |
| 增长量计算 | B、R | X | X = B × R |
| 增长率计算 | A、B | R | R = (A - B) / B |
| 增长率计算 | A、X | R | R = X / (A - X) |
| 现期计算 | B、R | A | A = B × (1 + R) |
| 现期计算 | B、X | A | A = B + X |

## 二、三大速算方法

### 方法 1：截位直除法（最常用）

**原理**：除法算式中，分子分母同时截取前几位（保留 2–3 位有效数字），大幅简化计算。

**操作步骤**：
1. 看选项差距：选项首位不同 → 截 2 位；首位相同第二位不同 → 截 3 位
2. 分母四舍五入截位（如 45678 → 46000 或 45700）
3. 分子可截可不截（视情况）
4. 直接做除法

**示例**：
- 题目：4287 / 32456 = ?
- 选项：A. 13.2%  B. 15.6%  C. 18.4%  D. 21.1%
- 选项首位都不同 → 截 2 位
- 4287 / 32456 ≈ 43 / 325 ≈ 43 / 32 × 1/10 ≈ 1.34 × 1/10 = 13.4%
- 选 A

**截位规则速记**：
- 选项差距大（首位不同）→ 分母截 2 位
- 选项差距小（首位同第二位不同）→ 分母截 3 位
- 选项差距极小（前两位都同）→ 分母截 4 位或精算

**易错点**：
- 分母截位后，分子尽量不截（保留精度）
- 当分母首位是 1 或 2 时，建议多截一位（因为 1 开头的数截位误差大）
- 负增长（下降）时，1 + R 变成 1 - |R|，别忘减

---

### 方法 2：415 份数法（算增长量专用）

**原理**：把基期量看作 100 份，增长率 R 对应的增长量是 R 份，现期量是 (100 + R) 份。通过份数关系快速求增长量。

**核心比例**：基期 : 增长量 : 现期 = 100 : R : (100 + R)

**适用场景**：已知现期量和增长率，求增长量。

**操作步骤**：
1. 把现期量除以 (100 + R) 份，得到每份的值
2. 增长量 = 每份的值 × R

**示例**：
- 题目：2023 年某地 GDP 为 8680 亿元，同比增长 8.4%，求增长量。
- 现期 : 增长量 = 108.4 : 8.4
- 每份 = 8680 / 108.4 ≈ 80（用截位直除：8680 / 108 ≈ 80）
- 增长量 = 80 × 8.4 ≈ 672 亿元
- 验证：8680 - 672 = 8008（基期），8008 × 8.4% ≈ 672 ✓

**415 份数法 vs 公式法**：
- 公式法：X = A × R / (1 + R) = 8680 × 8.4% / 108.4%，需要先乘再除，计算量大
- 415 份数法：先除后乘，且除法可截位，速度快 3 倍

**常见增长率对应的份数**（背熟提速）：

| 增长率 R | 基期 : 增长量 : 现期 | 现期÷份数≈ |
|---------|---------------------|-----------|
| 10% | 100 : 10 : 110 | 现期 / 110 |
| 20% | 100 : 20 : 120 | 现期 / 120 |
| 25% | 100 : 25 : 125 | 现期 / 125 |
| 50% | 100 : 50 : 150 | 现期 / 150 |
| 9.1% | 100 : 9.1 : 109.1 | 现期 / 109 |
| 16.7% | 100 : 16.7 : 116.7 | 现期 / 117 |
| 33.3% | 100 : 33.3 : 133.3 | 现期 / 133 |

**易错点**：
- 下降时 R 为负，现期 = 100 - |R| 份
- 增长率不是整数时，先估算每份再乘

---

### 方法 3：假设分配法（求基期专用）

**原理**：已知现期量和增长率，把现期量"分配"成基期和增长量两部分。

**操作步骤**：
1. 假设基期量 = 现期量（即假设增长量为 0）
2. 计算假设的增长量 = 假设基期 × R
3. 真实增长量 = 假设增长量 / (1 + R)
4. 真实基期 = 现期量 - 真实增长量

**示例**：
- 题目：2023 年产量 5460 万吨，同比增长 14%，求 2022 年产量。
- 假设 2022 年 = 5460，则假设增长量 = 5460 × 14% = 764.4
- 真实增长量 = 764.4 / 1.14 ≈ 671（截位直除：764 / 114 ≈ 6.7 × 100 = 670）
- 2022 年 = 5460 - 670 = 4790 万吨

**适用场景**：增长率在 5%–25% 之间时精度最高；增长率过大（>50%）误差大，改用截位直除。

**易错点**：
- 增长率越大，假设分配法误差越大
- 增长率为负时，假设分配法仍可用，但符号要小心

## 三、高频题型分类与解法

### 题型 1：基期量计算

**识别**："2022 年""上年同期""基期"等关键词。

**方法**：
- 已知现期 + 增长率 → B = A / (1 + R)，用截位直除
- 已知现期 + 增长量 → B = A - X，直接减

**示例**：2023 年 1—2 月，全国规模以上工业企业利润总额 8872 亿元，同比下降 22.9%。求 2022 年 1—2 月利润总额。
- B = 8872 / (1 - 22.9%) = 8872 / 0.771
- 截位直除：8872 / 0.771 ≈ 8870 / 0.77 ≈ 11500 亿元

---

### 题型 2：增长量计算

**识别**："增长了""净增""增加"等 + 具体数值（非百分比）。

**方法**：
- 已知现期 + 增长率 → 415 份数法
- 已知基期 + 增长率 → X = B × R，直接乘
- 已知现期 + 基期 → X = A - B，直接减

**增长量比较技巧**（"大大则大"）：
- 现期量大、增长率也大 → 增长量必然大
- 现期量大、增长率小（或反之）→ 用 X = A × R / (1+R) 比较，或用"分数比较法"

---

### 题型 3：增长率计算与比较

**识别**："增长了百分之几""增速""增幅"。

**计算**：R = (A - B) / B = X / B

**比较技巧**：
- 已知现期和基期 → 比较 (A/B - 1)，即比较 A/B
- 已知现期和增长量 → 比较 X / (A - X)，即比较 X/A（增长量 / 现期）
- 大幅下降时，比较 |R|，注意正负

**示例**：甲地 2023 年 GDP 12000 亿（上年 10000 亿），乙地 2023 年 GDP 8000 亿（上年 6500 亿），比较增速。
- 甲：12000/10000 - 1 = 20%
- 乙：8000/6500 - 1 ≈ 23%
- 乙增速更高

---

### 题型 4：比重计算与比较

**识别**："占""比重""比例"。

**核心公式**：
- 现期比重 = 部分 / 整体
- 基期比重 = (部分现期 / 整体现期) × [(1 + 整体增长率) / (1 + 部分增长率)]
- 比重升降 = 部分增长率 - 整体增长率（定性判断）
  - 部分增速 > 整体增速 → 比重上升
  - 部分增速 < 整体增速 → 比重下降
  - 部分增速 = 整体增速 → 比重不变

**比重差值（定量）**：
- 比重差 = (部分现期/整体现期) × [(部分R - 整体R) / (1 + 部分R)]
- 结果是百分点，不是百分比

**易错点**：
- 比重差单位是"百分点"，不是"%"
- 基期比重公式别记反：是整体 R 在分子，部分 R 在分母

---

### 题型 5：平均数计算

**识别**："平均""每""人均"。

**核心公式**：
- 现期平均数 = 总量 / 份数
- 基期平均数 = (总量/份数) × [(1 + 份数R) / (1 + 总量R)]
- 平均数增长率 = (总量R - 份数R) / (1 + 份数R)

**示例**：2023 年粮食产量 69541 万吨（+1.3%），播种面积 118969 万亩（+0.5%），求亩产增长率。
- 亩产 = 产量 / 面积
- 亩产增长率 = (1.3% - 0.5%) / (1 + 0.5%) = 0.8% / 1.005 ≈ 0.8%

---

### 题型 6：倍数计算

**识别**："是...的几倍""多几倍"。

**核心公式**：
- A 是 B 的几倍 → A / B
- A 比 B 多几倍 → A / B - 1
- 基期倍数 = (A/B) × [(1 + B的R) / (1 + A的R)]

**易错点**："是几倍"和"多几倍"差 1，"多几倍"= "是几倍" - 1。

---

### 题型 7：年均增长率

**识别**："年均增长""平均每年增长"。

**核心公式**：末期 / 初期 = (1 + 年均R)^n，n 为年份差

**估算方法**：
- n 较小（2–3 年）→ 直接开方
- n 较大 → 用"二项式展开"近似：(1+r)^n ≈ 1 + nr（r 较小时）
- 比较年均增速 → 比较 末期/初期 的 n 次方根，等价于比较 末期/初期（n 相同时）

**易错点**：
- n 是年份差，不是年份数（2018 到 2023 是 5 年，不是 6 年）
- "十三五"期间 n = 5（2016–2020）

---

### 题型 8：拉动增长率与贡献率

**识别**："拉动""贡献"。

**公式**：
- 拉动增长率 = 部分增长量 / 整体基期量
- 贡献率 = 部分增长量 / 整体增长量
- 关系：拉动增长率 = 贡献率 × 整体增长率

---

### 题型 9：综合分析题（最后一题）

**识别**：每篇材料最后一题，"以下说法正确/错误的是""能从材料推出的是"。

**解题策略**：
1. 优先验证计算量小的选项（如"比重""排名"类）
2. 验证顺序：D → B → C → A（经验上 D 正确概率略高，但不要迷信）
3. 找到一个正确/错误选项立即选，不验证其他
4. 选项中有"约""左右"→ 多半正确；有"超过""不足"→ 多验证

**易错点**：
- "超过"是 >，"不低于"是 ≥，"不足"是 <
- 时间范围要看清（"1—3 月"vs"3 月"）
- 单位要看清（万 / 亿 / % / ‰）

## 五、易错点汇总

| 易错点 | 说明 | 避坑方法 |
|-------|------|---------|
| 时间范围 | "1—3 月"vs"3 月"vs"一季度" | 划题干时间，对照材料 |
| 单位 | 万 / 亿 / 万吨 / 亿吨 / % / ‰ | 看材料表头单位 |
| 增长量 vs 增长率 | "增长 X"vs"增长 X%" | 看有无 % |
| 现期 vs 基期 | "2023 年"vs"2022 年" | 划时间词 |
| 比重差 vs 增长率差 | 百分点 vs 百分比 | 比重差是百分点 |
| 是几倍 vs 多几倍 | 差 1 | "多几倍"= "是几倍"-1 |
| 顺差 vs 逆差 | 出口 - 进口 vs 进口 - 出口 | 顺差 = 出口 > 进口 |
| 同比 vs 环比 | 与上年同期 vs 与上月 | 同比看年，环比看月 |

## 七、真题示例

### 示例 1（基期计算 + 截位直除）

**材料**：2023 年，全国棉花产量 561.8 万吨，同比下降 6.1%。

**问题**：2022 年全国棉花产量约为多少万吨？
A. 530  B. 580  C. 598  D. 620

**解析**：
- 题型：基期计算，B = A / (1 + R) = 561.8 / (1 - 6.1%) = 561.8 / 0.939
- 选项首位不同（5/5/5/6），第二位不同 → 截 2 位
- 561.8 / 0.939 ≈ 562 / 0.94 ≈ 598
- 选 C

**易错点**：下降时 1 + R = 1 - 6.1% = 0.939，别算成 1.061。

---

### 示例 2（增长量 + 415 份数法）

**材料**：2023 年，H 省地区生产总值 48810 亿元，同比增长 4.6%。

**问题**：2023 年 H 省 GDP 同比增长约多少亿元？
A. 1950  B. 2150  C. 2240  D. 2440

**解析**：
- 题型：增长量计算，已知现期 + 增长率 → 415 份数法
- 现期 : 增长量 = 104.6 : 4.6
- 每份 = 48810 / 104.6 ≈ 48810 / 105 ≈ 465（截位直除）
- 增长量 = 465 × 4.6 ≈ 2140
- 选 B

**易错点**：别用 X = A × R / (1+R) 直接算，计算量太大。

---

### 示例 3（比重比较 + 定性判断）

**材料**：2023 年，全国规模以上工业企业实现营业收入 159.4 万亿元，同比增长 1.1%。其中，私营企业实现营业收入 51.2 万亿元，同比增长 2.5%。

**问题**：2023 年私营企业营业收入占规模以上工业企业营业收入的比重，比上年同期：
A. 上升约 0.4 个百分点  B. 下降约 0.4 个百分点  C. 上升约 4 个百分点  D. 下降约 4 个百分点

**解析**：
- 题型：比重升降（定性 + 定量）
- 定性：部分 R（2.5%）> 整体 R（1.1%）→ 比重上升，排除 B、D
- 定量：比重差 = (51.2/159.4) × [(2.5% - 1.1%) / (1 + 2.5%)]
  - 51.2/159.4 ≈ 32%
  - 2.5% - 1.1% = 1.4%
  - 1.4% / 1.025 ≈ 1.37%
  - 32% × 1.37% ≈ 0.44 个百分点
- 选 A

**易错点**：比重差是百分点，不是百分比；别选 C（4 个百分点是把 32% 当成 100% 算的）。

---

### 示例 4（综合分析题）

**材料**：（略，假设为某省 2023 年经济数据）

**问题**：能够从上述资料中推出的是：
A. 2023 年该省 GDP 同比增速高于上年
B. 2023 年该省第三产业增加值占 GDP 比重超过 50%
C. 2023 年该省规模以上工业增加值同比增长超过 5%
D. 2022 年该省固定资产投资额低于 3 万亿元

**解题策略**：
1. 优先验证计算量小的：B（比重，一步除法）、A（增速比较，看材料有无上年增速）
2. 再验证 C、D（需要计算）
3. 找到一个能推出 / 不能推出的立即选

**易错点**：
- "高于上年"需要材料给上年增速，没给则不能推出
- "超过 50%"要精算，不能估算
- "低于 3 万亿"要算基期，注意时间

## 八、配套速记口诀

- **ABRX 互推**：现期除以一加率得基期，基期乘率得增量
- **截位直除**：选项差距看首位，不同截二同截三
- **415 份数**：现期除以一加率，得每份再乘增长率
- **比重升降**：部分大则升，部分小则降，差值看比重乘差率
- **综合分析**：先算简后算繁，找到一个就收手
