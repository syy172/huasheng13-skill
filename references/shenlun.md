# 申论（花生十三体系）

申论是公考主观题部分，分值 100 分，考试时间 150–180 分钟。花生十三体系的核心是 **"材料为王 + 题型套路 + 规范表达"**，强调答案必须从材料提炼，按题型套结构，用规范语言表达。

---

## 一、申论总体认知

### 1.1 申论本质

申论不是写作文，而是 **"阅读理解 + 信息提炼 + 规范表达"**。答案 80% 在材料里，关键在于"找得到、提得准、写得对"。

### 1.2 题型分类

| 题型 | 题量 | 分值 | 核心能力 |
|------|------|------|---------|
| 单一题（归纳概括） | 1–2 题 | 10–20 分 | 找点 + 概括 |
| 综合分析题 | 1 题 | 15–20 分 | 分析 + 逻辑 |
| 提出对策题 | 0–1 题 | 15–20 分 | 问题 + 对策 |
| 公文写作题 | 1–2 题 | 20–25 分 | 格式 + 内容 |
| 大作文 | 1 题 | 35–40 分 | 立意 + 结构 + 论证 |

### 1.3 评分标准

- **要点分**：每个关键词 / 要点 1–3 分
- **逻辑分**：分类合理、层次清晰 2–5 分
- **格式分**：公文格式正确 2–5 分
- **语言分**：表达规范、无语病 2–5 分
- **卷面分**：字迹工整 1–3 分

### 1.4 核心原则

1. **材料为王**：答案必须来自材料，不脱离材料
2. **关键词优先**：用材料原词，不随意替换
3. **分类作答**：同类合并，异类分开
4. **序号化**：用 ①②③ 或一、二、三标清
5. **字数控制**：不超字数，不少太多

## 二、单一题（归纳概括）

### 2.1 题型识别

**常见问法**：
- 概括...的问题 / 原因 / 影响 / 做法
- 归纳...的特点 / 表现 / 变化
- 总结...的经验 / 启示

### 2.2 解题步骤

#### Step 1：审题

- **作答对象**：概括什么（问题 / 原因 / 影响 / 做法）
- **范围**：根据哪则材料
- **字数**：不超过多少字
- **要求**：全面、准确、有条理

#### Step 2：找点

**找点方法**：
- 逐段阅读，标记与作答对象相关的内容
- 关注关联词后的内容（因为、所以、导致、使得）
- 关注标志性词汇（问题、原因、影响、措施）

**关键词定位**：
- 找问题：负面词（不足、缺乏、滞后、薄弱）
- 找原因：因果词（因为、由于、源于）
- 找影响：结果词（导致、使得、促进、推动）
- 找做法：动词（建立、完善、推进、加强）

#### Step 3：概括

**概括方法**：
- 删除修饰语，保留核心
- 同类合并
- 提炼关键词

**示例**：
- 原文："由于资金投入不足，导致基础设施建设滞后，影响了居民生活质量。"
- 概括为：①资金投入不足 ②基础设施建设滞后 ③影响居民生活质量

#### Step 4：分类

**分类方法**：
- 按主体分：政府 / 企业 / 社会 / 个人
- 按领域分：经济 / 政治 / 文化 / 社会 / 生态
- 按层面分：思想 / 制度 / 执行 / 监督
- 按时间分：过去 / 现在 / 未来

#### Step 5：书写

**书写格式**：
```
一是...。具体表现为：①...；②...；③...。
二是...。具体表现为：①...；②...；③...。
三是...。具体表现为：①...；②...；③...。
```

### 2.3 高频作答对象

#### 2.3.1 概括问题

**找负面表述**：不足、缺乏、滞后、薄弱、缺失、不到位、不完善。

**示例**：
- 原文："基层治理中存在人员配备不足、专业能力欠缺、经费保障不到位等问题。"
- 概括：①人员配备不足 ②专业能力欠缺 ③经费保障不到位

#### 2.3.2 概括原因

**找因果表述**：因为、由于、源于、归因于。

**分类**：
- 主观原因：思想认识、能力素质
- 客观原因：制度环境、资源条件
- 历史 vs 现实
- 内部 vs 外部

#### 2.3.3 概括影响

**分积极 / 消极**：
- 积极影响：促进、推动、提升、改善
- 消极影响：阻碍、破坏、加剧、恶化

#### 2.3.4 概括做法

**找动词**：建立、完善、推进、加强、创新、深化。

**概括为"动宾结构"**：
- 建立机制、完善制度、加强监管、创新方式

### 2.4 易错点

| 易错点 | 说明 |
|-------|------|
| 脱离材料 | 自己编答案，不在材料找 |
| 照抄原文 | 不概括，整段抄 |
| 不分类 | 要点堆砌，无层次 |
| 超字数 | 超出规定字数扣分 |
| 遗漏要点 | 材料有的没找到 |

## 三、综合分析题

### 3.1 题型分类

#### 3.1.1 词句理解题

**问法**：谈谈对"..."的理解 / 解释"...".

**答题结构**：
- 是什么（表层含义）
- 为什么（深层含义 / 原因）
- 怎么办（对策 / 启示）

#### 3.1.2 观点分析题

**问法**：对"...观点"进行评价 / 分析。

**答题结构**：
- 表态（认同 / 不认同 / 辩证）
- 理由（为什么）
- 对策（怎么办）

#### 3.1.3 比较分析题

**问法**：比较 A 和 B 的异同。

**答题结构**：
- 相同点
- 不同点
- 启示

#### 3.1.4 关系分析题

**问法**：分析 A 和 B 的关系。

**答题结构**：
- 各自含义
- 相互关系（对立统一、相互促进）
- 如何处理

### 3.2 解题步骤

1. **审题**：明确分析对象和角度
2. **定位**：找到材料中相关内容
3. **分析**：从多角度展开
4. **结构**：按"是什么—为什么—怎么办"组织
5. **书写**：逻辑清晰，层次分明

### 3.3 答题模板

#### 词句理解题模板

```
"..."是指...（表层含义）。具体表现为：①...；②...；③...。
其原因是...（深层分析）。
因此，我们应该...（对策启示）。
```

#### 观点分析题模板

```
此观点是正确 / 错误 / 片面的。
理由如下：①...；②...；③...。
因此，我们应该...（对策）。
```

## 四、提出对策题

### 4.1 题型识别

**问法**：
- 针对...问题，提出对策
- 如何解决...问题
- 请就...提出建议

### 4.2 对策来源

#### 4.2.1 直接对策（材料明示）

- 材料中的"应该""需要""必须"
- 材料中的经验做法
- 材料中的专家建议

#### 4.2.2 间接对策（反推）

- 问题反推：问题→解决该问题的对策
- 原因反推：原因→消除该原因的对策
- 教训反推：教训→避免该教训的对策

### 4.3 对策表述

**标准格式**：动宾结构 + 具体做法

**示例**：
- ❌ 加强监管
- ✅ 加强市场监管，建立健全监管机制，加大对违法行为的查处力度

### 4.4 对策维度（万能角度）

| 维度 | 对策方向 |
|------|---------|
| 思想 | 加强宣传教育，提高认识 |
| 制度 | 完善法律法规，健全制度 |
| 组织 | 加强领导，明确责任 |
| 人才 | 加强队伍建设，培养人才 |
| 资金 | 加大投入，保障经费 |
| 技术 | 加强研发，应用新技术 |
| 监管 | 加强监管，严格执法 |
| 监督 | 加强社会监督，畅通渠道 |

### 4.5 答题结构

```
针对...问题，提出以下对策：
一是...。具体措施：①...；②...。
二是...。具体措施：①...；②...。
三是...。具体措施：①...；②...。
```

## 五、公文写作题

### 5.1 公文类型

#### 5.1.1 法定公文

| 文种 | 适用场景 |
|------|---------|
| 通知 | 发布指示、布置工作 |
| 通报 | 表彰先进、批评错误 |
| 报告 | 汇报工作、反映情况 |
| 请示 | 请求指示、批准 |
| 函 | 不相隶属机关商洽工作 |
| 意见 | 提出见解和处理办法 |
| 批复 | 答复请示事项 |

#### 5.1.2 事务性公文

| 文种 | 适用场景 |
|------|---------|
| 讲话稿 | 领导讲话、会议发言 |
| 倡议书 | 发起倡议、号召行动 |
| 公开信 | 公开回应、说明情况 |
| 宣传稿 | 宣传政策、推广经验 |
| 调研报告 | 调查研究、提出建议 |
| 工作方案 | 部署工作、明确任务 |
| 编者按 | 介绍背景、点评内容 |

### 5.2 公文格式

**完整格式**：
```
标题（居中）
称呼（顶格，加冒号）：
  正文（开头 + 主体 + 结尾）
  结尾（此致 / 敬礼 / 特此通知等）
落款（右下）
日期（右下，落款下方）
```

**各部分说明**：

| 部分 | 要求 |
|------|------|
| 标题 | 发文机关 + 事由 + 文种 |
| 称呼 | 主送机关 / 对象 |
| 开头 | 发文缘由（为什么） |
| 主体 | 具体内容（是什么 / 怎么做） |
| 结尾 | 希望要求 / 结语 |
| 落款 | 发文机关 |
| 日期 | 年月日 |

### 5.3 常见公文模板

#### 5.3.1 通知

```
关于...的通知

XX单位：
  为...（目的），现就...事项通知如下：
  一、...
  二、...
  三、...
  请认真贯彻执行。

XX单位
XXXX年XX月XX日
```

#### 5.3.2 倡议书

```
关于...的倡议书

广大市民朋友们：
  ...（背景）。为此，我们倡议：
  一、...。
  二、...。
  三、...。
  让我们共同行动起来，...！

XX单位
XXXX年XX月XX日
```

#### 5.3.3 讲话稿

```
在...会议上的讲话

同志们：
  今天我们召开...会议，主要目的是...。
  一、认清形势，...。
  二、明确任务，...。
  三、狠抓落实，...。
  谢谢大家！
```

#### 5.3.4 工作方案

```
关于...的工作方案

  为...（目的），特制定本方案。
  一、指导思想
  ...
  二、工作目标
  ...
  三、主要任务
  （一）...
  （二）...
  四、保障措施
  （一）加强领导
  （二）明确责任
  （三）强化监督
```

### 5.4 公文写作要点

1. **格式正确**：标题、称呼、落款、日期齐全
2. **开头点题**：开门见山说明发文目的
3. **主体清晰**：分条列点，逻辑清楚
4. **语言规范**：用书面语，不用口语
5. **结尾得体**：根据文种用合适结语

## 六、大作文（议论文）

### 6.1 题型识别

**问法**：
- 结合给定资料，以"..."为题，写一篇文章
- 围绕"...主题"，自拟题目，写一篇议论文
- 请以"..."为话题，写一篇文章

**要求**：
- 字数 1000–1200 字
- 观点明确，论证有力
- 联系实际，结构完整

### 6.2 审题立意

#### 6.2.1 审题三步

**第一步：审题干**
- 题目给的关键词 / 主题
- 题目要求的角度

**第二步：审材料**
- 材料的主旨
- 材料的核心观点

**第三步：审要求**
- 字数
- 文体（议论文）
- 其他特殊要求

#### 6.2.2 立意方法

**方法 1：题干关键词法**
- 题干给关键词 → 围绕关键词立意
- 示例：题干"创新"→ 立意"以创新驱动发展"

**方法 2：材料主旨法**
- 找材料的核心观点
- 提炼为论点

**方法 3：角度切入法**
- 从是什么 / 为什么 / 怎么办切入
- 推荐从"怎么办"切入（对策型）

#### 6.2.3 立意要求

- **正确**：符合主流价值观
- **明确**：观点鲜明，不含糊
- **深刻**：有深度，不肤浅
- **新颖**：角度独特，不老套

### 6.3 结构安排

#### 6.3.1 五段三分式（最常用）

```
第一段：开头（150 字）—引出话题，提出论点
第二段：分论点一（250 字）
第三段：分论点二（250 字）
第四段：分论点三（250 字）
第五段：结尾（150 字）—总结全文，升华主题
```

#### 6.3.2 递进式

```
第一段：开头—提出论点
第二段：是什么—阐释概念
第三段：为什么—分析原因 / 意义
第四段：怎么办—提出对策
第五段：结尾—总结升华
```

#### 6.3.3 对比式

```
第一段：开头—提出论点
第二段：正面论证
第三段：反面论证
第四段：综合分析
第五段：结尾—总结
```

### 6.4 分论点生成

**方法**：从材料提炼，按维度展开。

**常见维度**：
- 主体维度：国家 / 社会 / 个人
- 领域维度：经济 / 政治 / 文化 / 社会 / 生态
- 层面维度：思想 / 制度 / 行动
- 时间维度：历史 / 现实 / 未来

**示例**：
- 主题：乡村振兴
- 分论点 1：产业兴旺是乡村振兴的基础（经济维度）
- 分论点 2：生态宜居是乡村振兴的关键（生态维度）
- 分论点 3：乡风文明是乡村振兴的灵魂（文化维度）

### 6.5 论证方法

#### 6.5.1 举例论证

**方法**：举具体事例证明论点。

**结构**：论点 + 例子 + 分析 + 小结

**示例**：
> 创新是引领发展的第一动力。以华为为例，面对技术封锁，华为坚持自主创新，研发投入连年增长，最终在 5G 领域取得领先地位。这充分说明，只有坚持创新，才能在激烈竞争中立于不败之地。

#### 6.5.2 道理论证

**方法**：用道理、原理证明论点。

**结构**：论点 + 道理 + 阐释

#### 6.5.3 对比论证

**方法**：正反对比证明论点。

**结构**：论点 + 正面 + 反面 + 分析

#### 6.5.4 引用论证

**方法**：引用名言、政策、数据证明论点。

**常用引用**：
- 古诗词
- 名人名言
- 政策文件
- 统计数据

### 6.6 开头写法

#### 6.6.1 背景式

```
当前，...（背景）。然而，...（问题）。因此，...（论点）。
```

#### 6.6.2 引用式

```
"...（名言）。"这句话告诉我们...（阐释）。在...背景下，...（论点）。
```

#### 6.6.3 事例式

```
...（事例）。这一事例启示我们...（论点）。
```

#### 6.6.4 设问式

```
面对...问题，我们该如何应对？答案在于...（论点）。
```

### 6.7 结尾写法

#### 6.7.1 总结式

```
综上所述，...（总结论点）。让我们...（号召）。
```

#### 6.7.2 展望式

```
展望未来，...（展望）。只要我们...，就一定能...（信心）。
```

#### 6.7.3 引用式

```
"...（名言）。"...（结合论点）。让我们...（号召）。
```

### 6.8 大作文易错点

| 易错点 | 说明 |
|-------|------|
| 偏题 | 立意脱离材料 / 题干 |
| 论点不清 | 观点含糊，不鲜明 |
| 结构混乱 | 分论点之间无逻辑 |
| 论证空洞 | 只有论点，无论证 |
| 套话连篇 | 用模板，无新意 |
| 字数不足 | 不到规定字数 |
| 脱离材料 | 不联系给定资料 |

## 九、真题示例

### 示例 1（归纳概括题）

**题目**：根据给定资料 1，概括当前基层治理中存在的主要问题。（15 分，不超过 200 字）

**材料节选**：
> "我们社区只有 5 个工作人员，要服务 2 万多居民，根本忙不过来。"社区主任无奈地说。同时，社区工作人员专业能力参差不齐，很多复杂的法律问题无法处理。此外，社区治理经费主要依靠财政拨款，来源单一，难以满足日益增长的治理需求。

**参考答案**：
```
一是人员配备不足。社区工作人员数量少，与服务对象数量不匹配。
二是专业能力欠缺。工作人员能力参差不齐，难以处理复杂问题。
三是经费保障单一。过度依赖财政拨款，来源渠道单一，难以满足需求。
```

### 示例 2（综合分析题）

**题目**：谈谈对"乡村振兴，关键在人"的理解。（20 分，不超过 300 字）

**参考答案**：
```
"乡村振兴，关键在人"是指人才是乡村振兴的核心要素和根本保障。
具体表现为：①产业振兴需要人才。发展现代农业、特色产业，需要懂技术、会经营的人才。②文化振兴需要人才。传承乡土文化、培育文明乡风，需要文化人才。③治理振兴需要人才。提升基层治理能力，需要高素质的基层干部。
然而，当前农村面临人才外流、老龄化等问题。
因此，要①加强人才培养，培育新型职业农民；②完善人才政策，吸引人才返乡；③优化人才环境，留住本土人才。
```

### 示例 4（大作文）

**题目**：以"创新引领未来"为题，写一篇议论文。（40 分，1000–1200 字）

**参考框架**：
```
标题：创新引领未来

开头（150 字）：
  当前，世界正经历百年未有之大变局，科技创新成为国际竞争的关键。面对新形势，唯有创新才能赢得未来。

分论点一（250 字）：创新是推动经济发展的核心动力。
  - 论证：举例华为、比亚迪等企业创新案例
  - 分析：创新提升竞争力

分论点二（250 字）：创新是破解发展难题的关键钥匙。
  - 论证：举例卡脖子技术突破
  - 分析：创新解决发展瓶颈

分论点三（250 字）：创新是实现民族复兴的必由之路。
  - 论证：从历史角度分析
  - 分析：创新关系国家命运

结尾（150 字）：
  总结三个分论点，号召全社会重视创新，为实现中华民族伟大复兴而奋斗。
```

- **对策题**：问题反推原因反推教训反推

- **大作文**：审题立意结构论证
- **规范表达**：动宾结构，书面语言
