## 六、成语辨析练习

### 练习 1

**题目**：面对复杂局面，我们需要______，不能______。
A. 标本兼治 / 扬汤止沸
B. 釜底抽薪 / 饮鸩止渴
C. 对症下药 / 因地制宜
D. 有的放矢 / 无的放矢

**解析**：
- 前后反义对应
- A：标本兼治（治标又治本）vs 扬汤止沸（治标不治本）→ 反义，符合
- B：釜底抽薪（根本解决）vs 饮鸩止渴（短视）→ 反义，但饮鸩止渴程度过重
- C：对症下药 vs 因地制宜 → 不是反义
- D：有的放矢 vs 无的放矢 → 反义，但语境不符

**答案**：A

### 练习 2

**题目**：传统文化需要______，在传承中创新。
A. 薪火相传
B. 代代相传
C. 一脉相承
D. 源远流长

**解析**：
- 语境：传承 + 创新
- A 薪火相传：一代代传授，符合
- B 代代相传：一代代传下去，符合
- C 一脉相承：从同一派别传下来，强调派别
- D 源远流长：历史悠久，不是传承动作
- "薪火相传"更有文化传承意味

**答案**：A

### 练习 3

**题目**：这项政策______，解决了群众反映强烈的问题。
A. 立竿见影
B. 行之有效
C. 卓有成效
D. 事半功倍

**解析**：
- 语境：政策有效，解决了问题
- A 立竿见影：见效快
- B 行之有效：实行起来有效
- C 卓有成效：成绩显著
- D 事半功倍：费力小收效大
- "解决了问题"强调成效，选 C

**答案**：C
