# 判断推理规律速查手册（花生十三体系）

本文件是 `panduan-tuili.md` 的补充，按子模块分类汇总所有规律、技巧，配识别标志，是判断推理的"规律字典"。

---

## 一、图形推理规律速查

### 1.1 规律优先级（必背）

```
元素相同 → 位置规律
元素相似 → 样式规律
元素不同 → 数量规律
都不行   → 属性规律
```

### 1.2 位置规律

| 规律 | 识别标志 | 操作 |
|------|---------|------|
| 平移 | 元素相同，位置变 | 看方向、步数 |
| 旋转 | 元素相同，角度变 | 看方向、角度 |
| 翻转 | 元素相同，镜像 | 看上下、左右 |
| 遍历 | 每种元素都出现 | 找缺失元素 |

#### 1.2.1 平移规律

| 类型 | 说明 |
|------|------|
| 直线平移 | 上下、左右移动 |
| 环形平移 | 沿边缘移动 |
| 步数规律 | 等差、恒定 |
| 方向规律 | 顺时针、逆时针 |

#### 1.2.2 旋转规律

| 类型 | 说明 |
|------|------|
| 角度 | 30°、45°、60°、90°、180° |
| 方向 | 顺时针、逆时针 |
| 交替 | 顺逆交替 |

### 1.3 样式规律

| 规律 | 识别标志 | 操作 |
|------|---------|------|
| 叠加 | 两图相加 | 看叠加规则 |
| 求同 | 找共同部分 | 保留相同 |
| 求异 | 找不同部分 | 去同存异 |
| 黑白叠加 | 黑白块叠加 | 找叠加规则 |

#### 1.3.1 黑白叠加规则（每题不同）

常见规则：
- 黑+黑=白，白+白=白，黑+白=黑
- 黑+黑=黑，白+白=白，黑+白=白
- 黑+黑=黑，白+白=黑，黑+白=白

### 1.4 数量规律

#### 1.4.1 数什么（优先级）

| 优先级 | 类别 | 识别标志 |
|--------|------|---------|
| 1 | 点 | 线条相交、切点 |
| 2 | 线 | 直线、曲线、一笔画 |
| 3 | 角 | 直角、锐角、钝角 |
| 4 | 面 | 封闭区域 |
| 5 | 素 | 元素种类、部分数 |

#### 1.4.2 点

| 类型 | 说明 |
|------|------|
| 交点 | 线条相交的点 |
| 切点 | 相切的点 |
| 顶点 | 多边形的角 |

#### 1.4.3 线

| 类型 | 说明 |
|------|------|
| 直线 | 直的线 |
| 曲线 | 弯曲的线 |
| 总线数 | 直线+曲线 |
| 一笔画 | 奇点 0 或 2 |

#### 1.4.4 一笔画判定

| 奇点数 | 能否一笔画 |
|--------|-----------|
| 0 | 能（任意起点） |
| 2 | 能（奇点出发） |
| 4, 6, 8... | 不能 |

**奇点**：从该点引出奇数条线（1, 3, 5, 7...条）

#### 1.4.5 角

| 类型 | 说明 |
|------|------|
| 直角 | 90° |
| 锐角 | <90° |
| 钝角 | >90° |
| 总角数 | 所有内角 |

#### 1.4.6 面

| 类型 | 说明 |
|------|------|
| 封闭面 | 封闭区域 |
| 半开面 | 半封闭 |
| 面积 | 面的大小 |

#### 1.4.7 素

| 类型 | 说明 |
|------|------|
| 种类数 | 不同元素种类 |
| 个数 | 元素总个数 |
| 部分数 | 连通部分 |

### 1.5 属性规律

| 规律 | 识别标志 | 操作 |
|------|---------|------|
| 对称性 | 图形对称 | 轴对称、中心对称 |
| 曲直性 | 全直/全曲 | 看线条 |
| 封闭性 | 全封闭/全开放 | 看是否封闭 |

#### 1.5.1 对称性

| 类型 | 说明 |
|------|------|
| 轴对称 | 沿轴对折重合 |
| 中心对称 | 绕中心旋转 180° 重合 |
| 既是轴又是中心 | 如矩形、圆 |

#### 1.5.2 对称轴数量

| 图形 | 对称轴数 |
|------|---------|
| 圆 | 无数 |
| 正方形 | 4 |
| 等边三角形 | 3 |
| 矩形 | 2 |
| 等腰三角形 | 1 |

### 1.6 立体图形

#### 1.6.1 折纸盒

| 方法 | 说明 |
|------|------|
| 相对面 | 展开图中间隔一个的面 |
| 相邻面 | 展开图中相邻的面 |
| 公共边 | 两面公共的边 |
| 公共点 | 三面公共的点 |

**相对面规律**：
- 一对面不能同时出现
- 展开图中"Z"字两端是相对面

#### 1.6.2 截面图

| 立体 | 可能截面 |
|------|---------|
| 正方体 | 三角形、四边形、五边形、六边形 |
| 圆柱 | 圆、矩形、椭圆 |
| 圆锥 | 三角形、圆、椭圆 |
| 球 | 圆 |

#### 1.6.3 三视图

| 视图 | 说明 |
|------|------|
| 主视图 | 从正面看 |
| 俯视图 | 从上面看 |
| 左视图 | 从左侧看 |

## 二、定义判断要素速查

### 2.1 定义八要素

| 要素 | 说明 | 关注词 |
|------|------|--------|
| 主体 | 谁做 | 谁、什么人 |
| 客体 | 对谁做 | 对什么、把什么 |
| 目的 | 为什么做 | 为了、目的是 |
| 方式 | 怎么做 | 通过、以...方式 |
| 条件 | 什么条件下 | 在...情况下 |
| 原因 | 为什么 | 因为、由于 |
| 结果 | 导致什么 | 导致、使得 |
| 性质 | 是什么 | 是、属于 |

### 2.2 解题步骤

1. **读定义**：找主体、客体、目的、方式
2. **画重点**：标记关键要素
3. **看选项**：逐一比对
4. **排除**：不符合要素的排除
5. **选择**：全部符合的选

### 2.3 常见定义类型

| 类型 | 例子 |
|------|------|
| 心理学 | 锚定效应、从众心理 |
| 经济学 | 沉没成本、机会成本 |
| 管理学 | 鲶鱼效应、破窗效应 |
| 法学 | 正当防卫、无因管理 |
| 社会学 | 社会化、角色冲突 |

## 三、类比推理关系速查

### 3.1 逻辑关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 全同 | 同一事物 | 番茄：西红柿 |
| 包含 | A 是 B 的一种 | 苹果：水果 |
| 组成 | A 是 B 的一部分 | 轮胎：汽车 |
| 交叉 | 部分重合 | 党员：干部 |
| 并列 | 同一类 | 苹果：香蕉 |
| 矛盾 | 非此即彼 | 生：死 |
| 反对 | 不止两类 | 红：绿 |

### 3.2 语义关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 近义 | 意思相近 | 美丽：漂亮 |
| 反义 | 意思相反 | 高：矮 |
| 褒贬 | 感情色彩 | 赞扬：批评 |

### 3.3 语法关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 主谓 | 主语+谓语 | 太阳：升起 |
| 动宾 | 动词+宾语 | 阅读：书籍 |
| 偏正 | 修饰+中心 | 美丽：花朵 |

### 3.4 对应关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 功能 | 事物与功能 | 杯子：喝水 |
| 材料 | 事物与材料 | 桌子：木头 |
| 职业 | 职业与场所 | 医生：医院 |
| 工具 | 职业与工具 | 教师：粉笔 |
| 因果 | 原因与结果 | 下雨：地湿 |
| 象征 | 事物与象征 | 鸽子：和平 |

### 3.5 二级辨析

当多个选项符合一级关系时，用二级辨析：

| 辨析 | 说明 |
|------|------|
| 方向 | A→B 还是 B→A |
| 程度 | 轻 vs 重 |
| 范围 | 大 vs 小 |
| 时间 | 先 vs 后 |
| 主体 | 人 vs 物 |
| 必然 | 必然 vs 偶然 |

## 四、逻辑判断规律速查

### 4.1 翻译推理

#### 4.1.1 翻译规则

| 关联词 | 翻译 |
|--------|------|
| 如果 A，那么 B | A → B |
| 只要 A，就 B | A → B |
| 只有 A，才 B | B → A |
| 除非 A，否则不 B | B → A |
| A 是 B 的前提 | B → A |
| A 是 B 的基础 | B → A |
| A 是 B 的必要条件 | B → A |

#### 4.1.2 推理规则

| 规则 | 说明 |
|------|------|
| 逆否等价 | A → B 等价于 ¬B → ¬A |
| 肯前必肯后 | A → B，A 真 → B 真 |
| 否后必否前 | A → B，¬B → ¬A |
| 肯后不确定 | A → B，B 真 → A 不确定 |
| 否前不确定 | A → B，¬A → B 不确定 |

#### 4.1.3 常见错误

| 错误 | 说明 |
|------|------|
| 肯后 | A → B，B → A（错） |
| 否前 | A → B，¬A → ¬B（错） |

### 4.2 真假推理

#### 4.2.1 矛盾关系

| 关系 | 说明 |
|------|------|
| A vs ¬A | 必有一真一假 |
| "所有"vs"有的不" | 必有一真一假 |
| "有的"vs"所有不" | 必有一真一假 |
| "A→B"vs"A且¬B" | 必有一真一假 |

#### 4.2.2 反对关系

| 关系 | 说明 |
|------|------|
| "所有"vs"所有不" | 至少一假 |
| "有的"vs"有的不" | 至少一真 |

#### 4.2.3 解题步骤

1. 找矛盾（一真一假）
2. 绕开矛盾看其他
3. 确定其他真假
4. 回到矛盾判断

### 4.3 加强削弱

#### 4.3.1 论证结构

```
论据（前提）→ 论点（结论）
```

#### 4.3.2 加强方法（力度排序）

| 方法 | 力度 | 说明 |
|------|------|------|
| 加强论点 | 最强 | 直接支持结论 |
| 加强论据 | 强 | 提供证据 |
| 建立联系 | 强 | 搭桥 |
| 排除他因 | 中 | 排除其他可能 |
| 举例加强 | 弱 | 举例子 |

#### 4.3.3 削弱方法（力度排序）

| 方法 | 力度 | 说明 |
|------|------|------|
| 否定论点 | 最强 | 直接否定结论 |
| 否定论据 | 强 | 否定前提 |
| 断开联系 | 强 | 拆桥 |
| 另有他因 | 中 | 指出其他原因 |
| 举例削弱 | 弱 | 反例 |

#### 4.3.4 常见陷阱

| 陷阱 | 说明 |
|------|------|
| 偷换概念 | 论点论据概念不一致 |
| 以偏概全 | 个别推一般 |
| 因果倒置 | 颠倒因果 |
| 滑坡谬误 | 夸大推论 |

### 4.4 朴素逻辑

#### 4.4.1 排序型

**方法**：画表格，填信息。

#### 4.4.2 匹配型

**方法**：列表，排除。

#### 4.4.3 真假型

**方法**：假设法，找矛盾。

#### 4.4.4 数独型

**方法**：排除法，唯一余数法。

## 五、规律速记口诀

### 5.1 图形推理
- 相同看位置，相似看样式，不同看数量，都不行看属性
- 一笔画奇点零或二
- 对称曲直和封闭
- 折纸盒看相对面

### 5.2 定义判断
- 主体客体目的方式，逐一比对
- 抓关键词，排除不符

### 5.3 类比推理
- 全同包含和组成，交叉并列矛盾反对
- 近义反义和褒贬，主谓动宾和偏正
- 功能材料职业工具，因果象征和对应
- 二级辨析方向程度范围

### 5.4 逻辑判断
- 翻译：前推后，后推前
- 推理：肯前否后，逆否等价
- 真假：找矛盾，绕开看
- 加强：否论点最强，举例最弱
- 削弱：否论点最强，另有他因中等
