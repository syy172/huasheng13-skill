# 错题本模板与复盘方法（花生十三体系）

本文件提供各模块错题本模板、复盘方法、归因分析，是备考过程的"错题管理工具"。

---

## 一、错题本总体认知

### 1.1 错题本的价值

- **避免重复犯错**：同类错题不再错
- **发现薄弱点**：找出知识盲区
- **巩固方法**：通过错题巩固解题方法
- **提升效率**：针对性复习，省时高效

### 1.2 错题本原则

1. **及时记录**：当天错题当天记
2. **分类整理**：按模块、题型分类
3. **归因分析**：分析错因，不只抄题
4. **定期回顾**：每周回顾，考前突击
5. **同类拓展**：举一反三，拓展同类

## 二、通用错题本模板

### 2.1 标准模板

```
【错题编号】XX-001
【模块】资料分析
【题型】基期计算
【日期】2024-XX-XX
【来源】XX 套题第 X 题

【题目】
（抄录题目，含材料）

【我的答案】B
【正确答案】C

【错因分析】
□ 知识盲区  □ 技巧不熟  □ 粗心大意  □ 时间不够
具体原因：截位直除时截位方向错，把下降算成上升

【正确解析】
题型：基期计算，已知现期 + 增长率（下降）
公式：基期 = 现期 / (1 + R)，R = -2.3%
计算：76858 / (1 - 2.3%) = 76858 / 0.977
截位：选项差距小，截 3 位
76858 / 0.977 ≈ 78660
答案：A

【易错点】
1. 下降时 1 + R = 1 - 2.3% = 0.977，别算成 1.023
2. 选项差距小要截 3 位

【同类拓展】
若问增长量，用 415 份数法
若问增长率，用 (现期-基期)/基期

【复习记录】
第 1 次复习：2024-XX-XX，已掌握
第 2 次复习：2024-XX-XX，已掌握
第 3 次复习：2024-XX-XX，已掌握
```

### 2.2 简化模板（适合大量错题）

```
【模块】资料分析 | 【题型】基期计算 | 【错因】技巧不熟
【题号】XX-001 | 【日期】XX-XX
【错因】截位方向错
【正解】下降用 1-R，截 3 位
【改进】重做 10 道截位题
```

## 四、错题归因分析

### 4.1 四类错因详解

#### 4.1.1 知识盲区

**表现**：
- 完全不会做
- 公式没记住
- 方法没学过

**对策**：
- 重新学习理论
- 记忆公式
- 学习方法
- 做基础题巩固

#### 4.1.2 技巧不熟

**表现**：
- 会做但慢
- 方法会用但容易错
- 速算不熟练

**对策**：
- 专项训练
- 限时练习
- 反复练习速算

#### 4.1.3 粗心大意

**表现**：
- 看错题
- 算错数
- 涂错卡
- 选错项

**对策**：
- 审题仔细
- 计算认真
- 涂卡检查
- 选项核对

#### 4.1.4 时间不够

**表现**：
- 题目会做但没时间
- 后面题目草草了事
- 涂卡来不及

**对策**：
- 调整做题顺序
- 学会放弃
- 提速训练
- 合理分配时间

### 4.2 归因统计表

| 模块 | 知识盲区 | 技巧不熟 | 粗心大意 | 时间不够 | 合计 |
|------|---------|---------|---------|---------|------|
| 资料分析 | | | | | |
| 数量关系 | | | | | |
| 言语理解 | | | | | |
| 判断推理 | | | | | |
| 申论 | | | | | |
| **合计** | | | | | |

**分析**：
- 知识盲区多 → 补理论
- 技巧不熟多 → 练速算
- 粗心多 → 练细心
- 时间不够多 → 调策略

## 五、复盘方法

### 5.1 套题复盘五步法

#### Step 1：对答案

- 逐题对答案
- 标记对错
- 统计各模块正确率

#### Step 2：析错因

- 每道错题分析错因
- 归入四类错因
- 记录到错题本

#### Step 3：找规律

- 错题集中在哪些模块
- 错题集中在哪些题型
- 错因主要是什么

#### Step 4：调策略

- 调整做题顺序
- 调整时间分配
- 调整选做策略

#### Step 5：定计划

- 针对薄弱模块专项
- 针对薄弱题型突破
- 制定下周计划

## 六、错题复习策略

### 6.1 艾宾浩斯复习法

| 复习时间 | 复习内容 |
|---------|---------|
| 当天 | 当天错题 |
| 第 2 天 | 昨天错题 |
| 第 7 天 | 上周错题 |
| 第 15 天 | 半月错题 |
| 第 30 天 | 上月错题 |
| 考前 | 所有错题 |

### 6.2 分类复习法

| 错因 | 复习频率 | 复习方法 |
|------|---------|---------|
| 知识盲区 | 每周 | 重学理论 + 做题 |
| 技巧不熟 | 每 3 天 | 重做 + 限时 |
| 粗心大意 | 每周 | 看错因 + 提醒 |
| 时间不够 | 每月 | 调策略 |

### 6.3 错题重做

**重做原则**：
- 遮住答案重做
- 完整写出过程
- 对比原解析
- 标记仍错的题

**重做周期**：
- 第 1 次：错后 1 周
- 第 2 次：错后 2 周
- 第 3 次：错后 1 月
- 第 4 次：考前
