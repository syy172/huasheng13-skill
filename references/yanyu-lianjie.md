# 言语理解与表达（花生十三体系）

言语理解是行测题量最大的模块（国考 40 题），分值占比高。花生十三体系的核心是 **"语境分析 + 关联词 + 中心句"三位一体**，强调先读题干结构再选答案，避免"凭语感"。

---

## 一、题型分类

| 题型 | 题量（国考） | 核心方法 |
|------|-------------|---------|
| 逻辑填空 | 20 题 | 语境分析 + 词义辨析 |
| 片段阅读 | 17–18 题 | 中心句识别 + 关联词 |
| 语句表达 | 2–3 题 | 话题一致 + 逻辑顺序 |
| 篇章阅读 | 0–10 题（副省） | 定位 + 同义替换 |

## 二、逻辑填空（20 题）

### 2.1 语境分析法（核心）

**原理**：填空题的答案一定在上下文中有"暗示"，找到暗示就能选对。

#### 2.1.1 对应关系（找暗示）

**四种对应**：

| 类型 | 标志词 | 例子 |
|------|--------|------|
| 反义对应 | 但是、然而、却 | "看似简单，___复杂"→ 实则 |
| 并列对应 | 和、与、以及、同时 | "美丽___智慧"→ 聪慧 |
| 递进对应 | 甚至、更、不仅...还 | "好，___更好"→ 尤 |
| 解释对应 | 即、也就是说、冒号 | "他___，即不说话"→ 沉默 |

**操作步骤**：
1. 找空格前后的关联词
2. 判断对应类型
3. 推导空格含义
4. 选最匹配的词

#### 2.1.2 搭配习惯

**常见固定搭配**（必背）：
- 汇聚 / 凝聚 力量
- 提高 / 提升 水平
- 推动 / 促进 发展
- 维护 / 捍卫 权益
- 履行 / 践行 承诺
- 传承 / 弘扬 文化
- 培育 / 厚植 精神
- 破解 / 化解 难题
- 激发 / 释放 活力
- 筑牢 / 夯实 基础

**易错搭配**：
- ❌ 提高范围 → ✅ 扩大范围
- ❌ 改善缺点 → ✅ 改正缺点
- ❌ 维持水平 → ✅ 保持水平
- ❌ 发扬缺点 → ✅ 改正缺点

### 2.2 词义辨析法

#### 2.2.1 语素差异法

**方法**：找两个词的不同语素，分析差异。

**示例**："遏制"vs"遏止"
- 遏制：制 = 控制，强调限制
- 遏止：止 = 停止，强调阻止
- "遏制势头"vs"遏止蔓延"

#### 2.2.2 程度轻重法

**方法**：根据语境判断需要轻还是重。

**常见程度梯度**：
- 轻到重：失望 → 绝望 → 绝望透顶
- 轻到重：批评 → 抨击 → 鞭挞
- 轻到重：阻碍 → 阻挠 → 破坏
- 轻到重：怀疑 → 质疑 → 否定

**选择原则**：
- 语境有"严重""极其"→ 选重的
- 语境温和 → 选轻的
- 不确定 → 选中间的

#### 2.2.3 感情色彩法

| 色彩 | 例子 |
|------|------|
| 褒义 | 赞扬、歌颂、卓越、杰出 |
| 贬义 | 妄想、煽动、狡猾、虚伪 |
| 中性 | 结果、影响、行为、现象 |

**选择原则**：褒义语境选褒义词，贬义语境选贬义词。

### 2.3 关联词法

**六大关联词**：

| 关系 | 关联词 | 作用 |
|------|--------|------|
| 转折 | 但是、然而、却、不过 | 重点在转折后 |
| 递进 | 而且、甚至、更 | 重点在递进后 |
| 因果 | 因此、所以、导致 | 重点在结果 |
| 条件 | 只有...才、只要...就 | 重点在条件 |
| 假设 | 如果...就、即使...也 | 重点在主句 |
| 并列 | 和、与、同时、既...又 | 无主次 |

**核心规律**：关联词后的内容是重点，填空要呼应重点。

### 2.4 高频成语辨析（必背 50 组）

#### 第一组：形容"多"
- **汗牛充栋**：形容书籍多（仅指书）
- **浩如烟海**：形容文献资料多
- **比比皆是**：形容到处都是
- **俯拾皆是**：形容多且容易得到
- **数不胜数**：形容极多

#### 第二组：形容"少"
- **寥寥无几**：形容非常少
- **凤毛麟角**：形容稀少而珍贵
- **屈指可数**：形容数量少
- **沧海一粟**：形容极其渺小

#### 第三组：形容"流行"
- **风靡一时**：形容一时流行
- **洛阳纸贵**：形容著作广泛流传
- **蔚然成风**：形容事物盛行成风气（褒义）
- **风靡一时**：形容流行（中性）

#### 第四组：形容"影响"
- **潜移默化**：暗中影响
- **耳濡目染**：受环境影响
- **润物无声**：默默影响（褒义）
- **言传身教**：用言行影响

#### 第五组：形容"努力"
- **殚精竭虑**：用尽精力（褒义）
- **处心积虑**：蓄意谋划（贬义）
- **苦心孤诣**：刻苦钻研
- **孜孜不倦**：勤奋不知疲倦

#### 第六组：形容"变化"
- **翻天覆地**：变化巨大
- **日新月异**：发展快
- **沧海桑田**：世事变化大
- **瞬息万变**：变化极快

#### 第七组：形容"相似"
- **如出一辙**：非常相似
- **大同小异**：大体相同
- **相差无几**：差别很小
- **毫厘不爽**：丝毫不差

#### 第八组：形容"独特"
- **独树一帜**：自成一家
- **别具一格**：另有一种风格
- **匠心独运**：独创性
- **标新立异**：提出新奇主张（中性偏贬）

### 2.5 逻辑填空解题步骤

1. **读题干**：先通读，把握大意
2. **找对应**：标出空格前后的关联词、对应词
3. **判类型**：反义 / 并列 / 递进 / 解释
4. **推含义**：根据对应推导空格应填的意思
5. **辨词义**：在候选词中选最匹配的
6. **代入验证**：把答案代入读一遍，看是否通顺

## 三、片段阅读（17–18 题）

### 3.1 中心理解题（最多）

**识别**："这段文字主要说明 / 讲的是 / 强调 / 意在"。

**核心方法**：找中心句。

#### 3.1.1 中心句位置

- **首句**：总分结构，首句是观点，后文是论据
- **尾句**：分总结构，前文是铺垫，尾句是结论
- **中间句**：承上启下，常在转折 / 递进后

#### 3.1.2 关联词定位

**重点在关联词后**：
- 转折后是重点（但是、然而）
- 递进后是重点（而且、甚至）
- 结论是重点（因此、所以）
- 对策是重点（应该、必须、需要）

**示例**：
> 传统媒体面临困境，但是新媒体的发展为信息传播提供了新机遇，我们应该抓住这一机遇，推动媒体融合。

- 转折词"但是"后是重点
- 对策"应该"后是核心
- 中心：抓住新媒体机遇，推动媒体融合

#### 3.1.3 主题词法

**方法**：找文段中反复出现的词，答案必须包含主题词。

**示例**：文段反复出现"乡村振兴"，答案必须含"乡村振兴"，不含的直接排除。

#### 3.1.4 对策优先原则

当文段提出问题 + 给对策时，答案优先选对策（"应该""需要""必须"）。

**错误选项特征**：
- 只提问题，不给对策
- 对策与文段不符
- 对策过于绝对

### 3.2 细节判断题

**识别**："以下说法正确 / 错误 / 符合 / 不符合的是"。

**解题方法**：逐项比对。

**常见陷阱**：

| 陷阱 | 说明 | 例子 |
|------|------|------|
| 偷换概念 | 改变主体 | 文段说"企业"，选项说"政府" |
| 偷换时态 | 改变时间 | 文段说"将"，选项说"已" |
| 偷换范围 | 扩大缩小 | 文段说"部分"，选项说"全部" |
| 偷换语气 | 改变程度 | 文段说"可能"，选项说"必然" |
| 无中生有 | 凭空捏造 | 文段没说的内容 |
| 正反颠倒 | 颠倒黑白 | 文段说"上升"，选项说"下降" |

**解题步骤**：
1. 看选项关键词
2. 回文段定位
3. 比对是否一致
4. 排除有陷阱的

### 3.3 标题填入题

**识别**："最适合做这段文字标题的是"。

**选择原则**：
1. 包含主题词
2. 概括主要内容
3. 吸引读者（生动 / 新颖）
4. 感情色彩一致

**优先级**：主题词 > 概括性 > 吸引力

### 3.4 语句填入题

**识别**："填入横线处最恰当的是"。

**解题方法**：
1. 看横线位置（首 / 中 / 尾）
2. 首句：概括后文
3. 中间：承上启下
4. 尾句：总结前文
5. 找前后文话题，保持一致

### 3.5 下文推断题

**识别**："接下来最可能讲的是"。

**解题方法**：
1. 看尾句话题
2. 顺承尾句往下讲
3. 排除前文已讲过的

## 四、语句表达（2–3 题）

### 4.1 语句排序题

**识别**：给 5–6 个句子，排正确顺序。

**解题方法**：

#### 4.1.1 看选项
- 选项首句有限制 → 排除不能做首句的
- 首句特征：引出话题、下定义、宏观表述
- 不能做首句：代词开头、关联词后半句、举例

#### 4.1.2 找捆绑
- 代词 + 前指：含"这""那""其"的句子，前一句必须出现指代对象
- 关联词配对："虽然...但是""不仅...而且"
- 顺序词："首先...其次...最后"

#### 4.1.3 看尾句
- 尾句特征：总结、对策、展望
- 排除不能做尾句的

#### 4.1.4 确定顺序
- 综合首尾 + 捆绑，确定唯一顺序

### 4.2 语句衔接题

参考 3.4 语句填入题方法。

## 五、篇章阅读（副省级 2 篇 10 题）

### 5.1 解题策略

1. **先看问题**：带着问题读文章
2. **定位原文**：找关键词对应段落
3. **同义替换**：答案多为原文同义替换
4. **细节题优先**：先做细节题，后做主旨题

### 5.2 题型分布

- 细节判断题（最多）
- 词句理解题
- 主旨概括题
- 标题填入题
- 语句填入题

## 七、易错点汇总

| 易错点 | 说明 |
|-------|------|
| 凭语感选 | 必须找语境对应 |
| 忽略关联词 | 关联词后是重点 |
| 主题词遗漏 | 答案必须含主题词 |
| 偷换陷阱 | 细节题逐项比对 |
| 排序首句错 | 代词、关联词后半句不能做首句 |
| 成语望文生义 | "不刊之论"= 不能更改的言论（褒义） |
| 程度轻重不匹配 | 语境重选重词，轻选轻词 |

## 八、真题示例

### 示例 1（逻辑填空—语境分析）

**题目**：传统文化不是博物馆里的陈列品，而是活态传承的有机体。它需要在传承中创新，在创新中传承，否则就会______。

A. 停滞不前  B. 故步自封  C. 日渐式微  D. 墨守成规

**解析**：
- 语境：不创新就会怎样？
- "否则"表反义对应，前文"活态传承"→ 反义是"失去活力"
- A 停滞不前：强调不前进
- B 故步自封：强调自满不求进步
- C 日渐式微：强调逐渐衰落（与"活态"反义）
- D 墨守成规：强调守旧
- "活态传承"反义是"衰落"，选 C

**答案**：C

### 示例 2（逻辑填空—成语辨析）

**题目**：在激烈的市场竞争中，企业只有不断创新，才能在______的商海中立于不败之地。

A. 波澜壮阔  B. 汹涌澎湃  C. 风起云涌  D. 惊涛骇浪

**解析**：
- 语境：商海竞争激烈，需要创新才能立足
- A 波澜壮阔：形容规模宏大（褒义，多形容景象）
- B 汹涌澎湃：形容声势浩大
- C 风起云涌：形容事物迅速发展
- D 惊涛骇浪：形容险恶的境遇（与"立于不败"呼应）
- "立于不败之地"暗示环境险恶，选 D

**答案**：D

### 示例 3（中心理解题）

**题目**：随着人工智能技术的发展，越来越多的工作岗位被机器取代。然而，技术进步也创造了大量新的就业机会，如数据标注师、AI 训练师等新兴职业不断涌现。因此，面对人工智能，我们不必过度恐慌，而应主动适应，提升技能。

这段文字意在说明：
A. 人工智能会取代大量工作岗位
B. 人工智能创造了新的就业机会
C. 应对人工智能应主动适应、提升技能
D. 人工智能技术发展迅速

**解析**：
- 关联词："然而"（转折）→ 重点在后
- "因此"（因果）→ 结论是重点
- 结论："不必恐慌，应主动适应，提升技能"
- 对应 C

**答案**：C

---

### 示例 4（细节判断题）

**题目**：根据文段，以下说法正确的是：
A. 人工智能将完全取代人类工作
B. 新兴职业的出现缓解了就业压力
C. 所有人都应该学习人工智能技术
D. 人工智能技术发展已经成熟

**解析**：
- A "完全取代"过于绝对，文段说"越来越多"不是"完全"
- B 文段提到新兴职业涌现，可推出缓解就业压力
- C "所有人"过于绝对
- D "已经成熟"文段未提

**答案**：B

### 示例 5（语句排序题）

**题目**：① 这种现象引起了社会的广泛关注
② 近年来，共享单车乱停乱放问题日益严重
③ 一些城市开始出台管理办法
④ 同时，企业也加强了运营管理
⑤ 共享单车行业逐渐走向规范

A. ②①③④⑤  B. ②①④③⑤  C. ①②③④⑤  D. ②③①④⑤

**解析**：
- 首句：①"这种现象"含代词，不能做首句 → 排除 C
- ②引出话题"共享单车乱停乱放"，适合做首句
- ①"这种现象"指代②的现象 → ②①捆绑
- ③"出台管理办法"和④"企业加强管理"是并列应对
- ⑤"走向规范"是结果，做尾句
- 顺序：②①③④⑤

**答案**：A

## 九、配套速记口诀

- **填空找对应**：转折反义、并列近义、递进加深、解释同义
- **中心找关联**：转折后、递进后、结论是重点
- **主题词必含**：反复出现是主题，答案必含主题词
- **细节防偷换**：概念、时态、范围、语气、有无、正反
- **排序看首尾**：代词不做首，总结做尾句
- **成语辨析**：语素差异、程度轻重、感情色彩
