## 一、截位直除法进阶

### 1.1 截位规则详解

#### 1.1.1 看选项决定截几位

| 选项特征 | 截位 | 示例 |
|---------|------|------|
| 首位不同 | 分母截 2 位 | 12% / 23% / 34% / 45% |
| 首位同，第二位不同 | 分母截 3 位 | 12.1% / 12.5% / 12.8% / 13.2% |
| 前两位同，第三位不同 | 分母截 4 位 | 12.11% / 12.15% / 12.18% / 12.22% |
| 前三位同 | 精算或特殊方法 | 极少见 |

#### 1.1.2 截位方向

**分母截位**：四舍五入（最常用）
- 45678 → 46000（截 2 位）
- 45678 → 45700（截 3 位）

**分子截位**：视情况
- 分子小（< 分母 1/10）→ 不截，保留精度
- 分子大 → 可截，但一般不截

#### 1.1.3 特殊数字处理

| 原数 | 截 2 位 | 截 3 位 | 说明 |
|------|---------|---------|------|
| 19xxx | 19 | 19x | 首位是 1，截 2 位误差大，建议截 3 位 |
| 99xxx | 99 | 99x | 接近 100，可当 100 算 |
| 50xxx | 50 | 50x | 整数，好算 |
| 33xxx | 33 | 33x | 可当 1/3 算 |

### 1.2 截位直除的变体

#### 1.2.1 分子分母同截

**适用**：分子分母都很大，且选项差距大。

**示例**：
- 456789 / 123456 ≈ 46 / 12 ≈ 3.83
- 选项：3.5 / 3.8 / 4.1 / 4.4 → 选 3.8

#### 1.2.2 只截分母

**适用**：分子较小，需保留精度。

**示例**：
- 1234 / 45678 ≈ 1234 / 46000 ≈ 0.0268
- 选项：2.5% / 2.7% / 2.9% / 3.1% → 选 2.7%

#### 1.2.3 凑整法

**适用**：分母接近整数。

**示例**：
- 4567 / 998 ≈ 4567 / 1000 = 4.567
- 选项：4.3 / 4.5 / 4.7 / 4.9 → 选 4.5

### 1.3 截位直除练习

**练习 1**：34567 / 89234 = ?
- 选项：A. 35%  B. 39%  C. 43%  D. 47%
- 首位不同 → 截 2 位
- 34567 / 89234 ≈ 35 / 89 ≈ 0.393
- 选 B

**练习 2**：123456 / 456789 = ?
- 选项：A. 25.0%  B. 27.0%  C. 29.0%  D. 31.0%
- 首位同（2），第二位不同 → 截 3 位
- 123456 / 456789 ≈ 123 / 457 ≈ 0.269
- 选 B

**练习 3**：56789 / 234567 = ?
- 选项：A. 22.1%  B. 24.2%  C. 26.3%  D. 28.4%
- 首位同（2），第二位不同 → 截 3 位
- 56789 / 234567 ≈ 568 / 2346 ≈ 0.242
- 选 B

## 二、415 份数法进阶

### 2.1 适用场景

**核心场景**：已知现期 + 增长率，求增长量。

**公式**：
- 现期 : 增长量 = (100 + R) : R
- 每份 = 现期 / (100 + R)
- 增长量 = 每份 × R

### 2.2 常见增长率与份数对应

| 增长率 R | 现期 : 增长量 | 简化比 |
|---------|--------------|--------|
| 10% | 110 : 10 | 11 : 1 |
| 20% | 120 : 20 | 6 : 1 |
| 25% | 125 : 25 | 5 : 1 |
| 33% | 133 : 33 | 4 : 1 |
| 50% | 150 : 50 | 3 : 1 |
| 66% | 166 : 66 | 5 : 2 |
| 100% | 200 : 100 | 2 : 1 |

### 2.3 415 份数法操作步骤

**Step 1**：确定增长率 R
**Step 2**：写出比例 现期 : 增长量 = (100+R) : R
**Step 3**：化简比例（如 110:10 → 11:1）
**Step 4**：算每份 = 现期 / (100+R)
**Step 5**：增长量 = 每份 × R

### 2.4 415 份数法示例

**示例 1**：现期 48810，增长率 4.6%，求增长量
- 比例：104.6 : 4.6
- 每份 = 48810 / 104.6 ≈ 48810 / 105 ≈ 465
- 增长量 = 465 × 4.6 ≈ 2140

**示例 2**：现期 12000，增长率 25%，求增长量
- 比例：125 : 25 = 5 : 1
- 每份 = 12000 / 5 = 2400
- 增长量 = 2400 × 1 = 2400

**示例 3**：现期 8000，增长率 33%，求增长量
- 比例：133 : 33 = 4 : 1
- 每份 = 8000 / 4 = 2000
- 增长量 = 2000 × 1 = 2000

### 2.5 415 份数法变体

#### 2.5.1 已知基期 + 增长率，求增长量

**方法**：增长量 = 基期 × R（直接乘，不用 415）

**示例**：基期 4000，增长率 25%，增长量 = 4000 × 25% = 1000

#### 2.5.2 已知现期 + 增长量，求增长率

**方法**：R = 增长量 / (现期 - 增长量)

**示例**：现期 5000，增长量 1000，R = 1000 / 4000 = 25%

### 2.6 415 份数法练习

**练习 1**：现期 6300，增长率 5%，求增长量
- 比例：105 : 5 = 21 : 1
- 每份 = 6300 / 21 = 300
- 增长量 = 300 × 1 = 300

**练习 2**：现期 9600，增长率 20%，求增长量
- 比例：120 : 20 = 6 : 1
- 每份 = 9600 / 6 = 1600
- 增长量 = 1600 × 1 = 1600

**练习 3**：现期 7500，增长率 50%，求增长量
- 比例：150 : 50 = 3 : 1
- 每份 = 7500 / 3 = 2500
- 增长量 = 2500 × 1 = 2500

## 三、假设分配法进阶

### 3.1 适用场景

**核心场景**：已知基期 + 增长率，求现期（或反过来）。

**原理**：把基期分成 100 份，现期 = 100 + R 份。

### 3.2 假设分配法操作步骤

**Step 1**：设基期 = 100 份
**Step 2**：现期 = 100 + R 份
**Step 3**：每份 = 实际基期 / 100
**Step 4**：现期 = 每份 × (100 + R)

### 3.3 假设分配法示例

**示例 1**：基期 5000，增长率 8%，求现期
- 设基期 = 100 份，现期 = 108 份
- 每份 = 5000 / 100 = 50
- 现期 = 50 × 108 = 5400

**示例 2**：基期 8000，增长率 12.5%，求现期
- 设基期 = 100 份，现期 = 112.5 份
- 每份 = 8000 / 100 = 80
- 现期 = 80 × 112.5 = 9000

### 3.4 假设分配法 vs 截位直除

| 方法 | 适用 | 优势 |
|------|------|------|
| 假设分配 | 已知基期 + R，求现期 | 避免除法，用乘法 |
| 截位直除 | 已知现期 + R，求基期 | 直接除，简单 |

**记忆**：基期 → 现期用假设分配（乘法），现期 → 基期用截位直除（除法）。

### 3.5 假设分配法练习

**练习 1**：基期 6000，增长率 15%，求现期
- 每份 = 6000 / 100 = 60
- 现期 = 60 × 115 = 6900

**练习 2**：基期 4500，增长率 20%，求现期
- 每份 = 4500 / 100 = 45
- 现期 = 45 × 120 = 5400

## 四、其他速算技巧

### 4.1 分数比较法

**适用**：比较多个分数大小。

**方法**：
1. 通分（分母相同）
2. 或交叉相乘（a/b vs c/d → ad vs bc）
3. 或找中间值（如 1/2、1/3）

**示例**：比较 3/7、4/9、5/11
- 3/7 ≈ 0.428
- 4/9 ≈ 0.444
- 5/11 ≈ 0.454
- 5/11 > 4/9 > 3/7

### 4.2 首数法

**适用**：选项首位不同时，只算首位。

**示例**：4567 / 892 = ?
- 选项：A. 3  B. 4  C. 5  D. 6
- 首位：45 / 89 ≈ 0.5 → 5
- 选 C

### 4.3 尾数法

**适用**：加减法，选项尾数不同。

**示例**：4567 + 3456 + 2345 = ?
- 选项：A. 10368  B. 10369  C. 10370  D. 10371
- 尾数：7 + 6 + 5 = 18 → 尾数 8
- 选 A

### 4.4 范围排除法

**适用**：估算结果范围，排除选项。

**示例**：4567 × 892 = ?
- 选项：A. 3xxx  B. 4xxx  C. 5xxx  D. 6xxx
- 4567 × 892 ≈ 4500 × 900 = 4050000 → 4xxx
- 选 B

### 4.5 特殊分数转化

| 分数 | 小数 | 百分比 |
|------|------|--------|
| 1/2 | 0.5 | 50% |
| 1/3 | 0.333 | 33.3% |
| 1/4 | 0.25 | 25% |
| 1/5 | 0.2 | 20% |
| 1/6 | 0.167 | 16.7% |
| 1/7 | 0.143 | 14.3% |
| 1/8 | 0.125 | 12.5% |
| 1/9 | 0.111 | 11.1% |

**应用**：增长率接近这些值时，直接用分数算。

## 五、综合速算策略

### 5.1 速算方法选择流程

```
看题目类型
├─ 求基期（已知现期+R）→ 截位直除
├─ 求现期（已知基期+R）→ 假设分配
├─ 求增长量（已知现期+R）→ 415份数
├─ 求增长量（已知基期+R）→ 直接乘
├─ 求增长率 → 增量/基期
├─ 比较大小 → 分数比较法
└─ 综合计算 → 多方法结合
```

### 5.2 速算训练计划

#### 第一周：截位直除
- 每天 20 道截位直除题
- 重点练截位判断

#### 第二周：415 份数
- 每天 20 道 415 份数题
- 重点练比例化简

#### 第三周：假设分配
- 每天 20 道假设分配题
- 重点练乘法

#### 第四周：综合训练
- 每天 2 篇资料分析
- 综合运用三种方法

### 5.3 速算常见错误

| 错误 | 原因 | 对策 |
|------|------|------|
| 截位方向错 | 没看清选项差距 | 先看选项再截位 |
| 415 比例错 | 没化简 | 先化简再算 |
| 假设分配乘错 | 乘法不熟 | 多练乘法 |
| 单位换算错 | 没注意单位 | 先统一单位 |
| 正负号错 | 下降算成上升 | 看清"增长""下降" |

## 六、速算口诀总集

- **截位直除**：选项差距看首位，不同截二同截三
- **415 份数**：现期除以一加率，得每份再乘增长率
- **假设分配**：基期设百份，现期百加率
- **分数比较**：通分交叉中间值
- **首数法**：首位不同只算首
- **尾数法**：加减算尾数
- **特殊分数**：1/3 1/4 1/5 1/6 1/7 1/8 1/9 要记牢
