# 判断推理（花生十三体系）

判断推理是行测技巧性最强的模块，共 40 题（国考），分四大子模块：图形推理（10 题）、定义判断（10 题）、类比推理（10 题）、逻辑判断（10 题）。花生十三体系强调 **"规律体系化 + 选项排除"**，把感性的"看图"变成理性的"套规律"。

---

## 一、图形推理（10 题）

### 1.1 总体解题流程

```
看图 → 判断题型（规律类 / 分组类 / 立体类）→ 套规律体系 → 选答案
```

### 1.2 规律类图形推理（6–7 题）

#### 1.2.1 位置规律

**识别**：图形组成元素相同，只是位置不同。

| 规律 | 说明 | 示例 |
|------|------|------|
| 平移 | 元素上下左右移动 | 黑块每次右移 1 格 |
| 旋转 | 顺时针 / 逆时针旋转 | 每次顺时针转 90° |
| 翻转 | 上下 / 左右翻转 | 左右镜像 |
| 遍历 | 每种元素都出现一次 | 每行含所有形状 |

#### 1.2.2 样式规律

**识别**：图形组成元素相似。

| 规律 | 说明 |
|------|------|
| 加减同异 | 叠加求同 / 求异 |
| 黑白叠加 | 黑+黑=白，白+白=白，黑+白=黑（每题规则不同） |
| 求同 | 保留相同部分 |
| 求异 | 去掉相同部分，保留不同部分 |

#### 1.2.3 数量规律（高频）

**识别**：图形组成元素不同，无明显位置 / 样式规律。

**数什么**（按优先级）：

| 优先级 | 数量类 | 识别标志 |
|--------|--------|---------|
| 1 | 点 | 线条相交、切点 |
| 2 | 线 | 直线、曲线、一笔画 |
| 3 | 角 | 直角、锐角、钝角 |
| 4 | 面 | 封闭区域数 |
| 5 | 素 | 元素种类数、部分数 |

**一笔画问题**（高频考点）：
- 判定：奇点数 = 0 或 2 → 可一笔画
- 奇点：从该点引出奇数条线
- 常见一笔画图形：圆、三角形、五角星

**数量规律常见形式**：
- 等差数列（1, 2, 3, 4, 5）
- 对称（1, 2, 3, 2, 1）
- 恒定（都是 3）
- 运算（前两个之和 = 第三个）

#### 1.2.4 属性规律

**识别**：图形无明显数量规律。

| 属性 | 说明 |
|------|------|
| 对称性 | 轴对称、中心对称 |
| 曲直性 | 全直、全曲、曲直结合 |
| 封闭性 | 全封闭、全开放、半封闭 |

#### 1.2.5 规律优先级（必背）

**"组成相同看位置，组成相似看样式，组成不同看数量，数量不行看属性"**

```
元素相同 → 位置规律（平移、旋转、翻转、遍历）
元素相似 → 样式规律（叠加、求同求异）
元素不同 → 数量规律（点线角面素）
都不行   → 属性规律（对称、曲直、封闭）
```

### 1.3 分组类图形推理（2–3 题）

**识别**：给 6 个图形，分成两组，每组有共同规律。

**解题方法**：
1. 找一个能把 6 图分两组的规律
2. 常见分组依据：对称 / 曲直 / 封闭 / 数量奇偶

### 1.4 立体类图形推理（1–2 题）

#### 1.4.1 折纸盒（六面体）

**方法**：相对面法 + 相邻面法。

**相对面规律**：
- 一对面不能同时出现
- 相对面在展开图中：相间排列或呈 Z 字形两端

**相邻面法**：
- 找三个相邻面的相对位置
- 用"时针法"判断（顺时针 / 逆时针）

**公共边法**：
- 两个面的公共边在折叠后不变

#### 1.4.2 截面图

**常见截面**：
- 正方体：三角形、四边形、五边形、六边形
- 圆柱：圆、矩形、椭圆
- 圆锥：三角形、椭圆、抛物线

#### 1.4.3 三视图

- 主视图：从前向后看
- 俯视图：从上向下看
- 左视图：从左向右看

### 1.5 图形推理高频考点

1. 一笔画（每年 1–2 题）
2. 封闭区域数（每年 1–2 题）
3. 对称性（每年 1–2 题）
4. 平移 / 旋转（每年 1–2 题）
5. 黑白叠加（每年 1 题）
6. 折纸盒（每年 1–2 题）
7. 元素种类数（每年 1 题）

## 二、定义判断（10 题）

### 2.1 核心方法：要素法

**原理**：每个定义都由若干要素构成，把要素拆出来，逐一比对选项。

### 2.2 八大要素

| 要素 | 说明 | 例子 |
|------|------|------|
| 主体 | 谁做 | 政府、企业、个人 |
| 客体 | 对谁做 | 消费者、产品 |
| 目的 | 为了什么 | 促进发展、保护权益 |
| 方式 | 怎么做 | 通过...手段 |
| 条件 | 在什么情况下 | 在...时 |
| 原因 | 因为什么 | 由于... |
| 结果 | 产生什么 | 导致... |
| 性质 | 是什么 | 是一种...行为 |

### 2.3 解题步骤

1. **读定义**：标出主体、客体、目的、方式等要素
2. **看问法**：属于 / 不属于
3. **逐项比对**：每个选项对照要素
4. **排除**：有一个要素不符即排除

### 2.4 常见题型

#### 2.4.1 单定义判断（7–8 题）

**示例**：
> 定义：沉没成本是指已经发生且无法收回的支出，与当前决策无关。

**要素拆解**：
- 主体：支出（已发生）
- 条件：无法收回
- 性质：与当前决策无关

**选项比对**：
- A. 小明买了电影票，看了一半觉得不好看，但想着票钱已花就继续看 → 符合（票钱是沉没成本，但小明受其影响）
- B. 企业投资某项目亏损，决定追加投资挽回 → 符合（前期投资是沉没成本，但企业受其影响）
- C. ... → 逐项比对

#### 2.4.2 多定义判断（2–3 题）

**方法**：先看问哪个定义，重点读该定义，其他定义略读。

### 2.5 易错点

| 易错点 | 说明 |
|-------|------|
| 主体不符 | 定义是"政府"，选项是"企业" |
| 客体不符 | 定义是"对消费者"，选项是"对员工" |
| 方式不符 | 定义是"通过法律手段"，选项是"通过行政手段" |
| 目的不符 | 定义是"为了盈利"，选项是"为了公益" |
| 忽略"不" | "不属于"题型，要选不符合的 |

## 三、类比推理（10 题）

### 3.1 核心方法：关系层级法

**原理**：词与词之间的关系有层级，先判大类，再判小类，最后用选项排除。

### 3.2 六大关系

#### 3.2.1 语义关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 近义 | 意思相近 | 开心：快乐 |
| 反义 | 意思相反 | 黑：白 |
| 象征 | A 象征 B | 鸽子：和平 |
| 比喻 | A 比喻 B | 红颜：美女 |

#### 3.2.2 逻辑关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 全同 | A 就是 B | 番茄：西红柿 |
| 并列 | 同类不同种 | 苹果：香蕉 |
| 交叉 | 部分重合 | 党员：干部 |
| 包含 | A 包含 B | 水果：苹果 |
| 组成 | A 是 B 的一部分 | 树：树根 |

**包含 vs 组成**（易混）：
- 包含：A 是 B 的一种（水果含苹果）
- 组成：A 是 B 的一部分（树含树根）

#### 3.2.3 对应关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 功能 | A 的功能是 B | 笔：写字 |
| 材料 | A 由 B 制成 | 桌子：木头 |
| 职业 | A 的职业是 B | 医生：治病 |
| 因果 | A 导致 B | 地震：海啸 |
| 工具 | A 用 B | 农民：锄头 |
| 配套 | A 配 B | 杯子：杯盖 |
| 顺承 | A 后是 B | 起跑：冲刺 |

#### 3.2.4 语法关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 主谓 | 主语+谓语 | 太阳：升起 |
| 动宾 | 动词+宾语 | 阅读：书籍 |
| 偏正 | 修饰+中心 | 美丽：花朵 |

#### 3.2.5 程度关系

| 关系 | 说明 | 例子 |
|------|------|------|
| 程度加深 | A 比 B 更... | 喜欢：热爱 |
| 程度减弱 | A 比 B 更弱 | 愤怒：不满 |

#### 3.2.6 词性关系

| 关系 | 说明 |
|------|------|
| 同词性 | 都是名词 / 动词 / 形容词 |
| 异词性 | 一名一动等 |

### 3.3 解题步骤

1. **判大类**：题干是语义 / 逻辑 / 对应 / 语法
2. **判小类**：进一步细分（如并列、包含、组成）
3. **看方向**：A→B 还是 B→A
4. **选项排除**：用二级辨析（词性、感情色彩、具体抽象）

### 3.4 二级辨析（当多个选项关系相同时）

| 辨析维度 | 说明 |
|---------|------|
| 词性 | 都是名词还是一动一名 |
| 感情色彩 | 褒贬一致 |
| 具体 / 抽象 | 都是具体事物还是一具体一抽象 |
| 必然 / 或然 | 必然关系还是可能关系 |
| 自然 / 人造 | 自然物还是人造物 |

### 3.5 三词型类比

**方法**：先看前两词关系，再看与第三词关系。

**示例**：医生：医院：治病
- 医生在医院治病
- 对应：教师：学校：教书

## 四、逻辑判断（10 题）

### 4.1 翻译推理（2–3 题）

**识别**：含"如果...那么""只有...才""只要...就"等逻辑关联词。

#### 4.1.1 翻译规则

| 关联词 | 翻译 | 记忆 |
|--------|------|------|
| 如果 A 那么 B | A → B | 前推后 |
| 只要 A 就 B | A → B | 前推后 |
| 只有 A 才 B | B → A | 后推前 |
| 除非 A 否则 B | ¬A → B | 否一推一 |
| A 是 B 的必要条件 | B → A | 后推前 |
| A 是 B 的充分条件 | A → B | 前推后 |

#### 4.1.2 推理规则

**逆否等价**：A → B 等价于 ¬B → ¬A

**传递推理**：A → B，B → C，则 A → C

**德摩根**：
- ¬(A 且 B) = ¬A 或 ¬B
- ¬(A 或 B) = ¬A 且 ¬B

#### 4.1.3 常见陷阱

- **肯后肯前**：A → B，B 成立 → A 成立（错！）
- **否前否后**：A → B，¬A → ¬B（错！）
- 只有逆否命题才等价

**示例**：
- "如果下雨，我就带伞"→ 下雨 → 带伞
- 带伞了 → 一定下雨吗？不一定（肯后错）
- 没下雨 → 一定不带伞吗？不一定（否前错）
- 没带伞 → 一定没下雨（逆否，对）

### 4.2 真假推理（1–2 题）

**识别**：几人说几话，有真有假。

#### 4.2.1 四种关系

| 关系 | 说明 |
|------|------|
| 矛盾 | 必一真一假 |
| 反对 | 至少一假（上反对）/ 至少一真（下反对） |
| 从属 | 全称真→特称真，特称假→全称假 |
| 下反对 | 至少一真 |

#### 4.2.2 常见矛盾

- 所有是 vs 有些非
- 所有非 vs 有些是
- 某个是 vs 某个非
- A 且 B vs ¬A 或 ¬B
- A 或 B vs ¬A 且 ¬B
- A → B vs A 且 ¬B

#### 4.2.3 解题步骤

1. 找矛盾（或反对）
2. 真假在矛盾中
3. 矛盾外的都确定
4. 推矛盾内

### 4.3 加强削弱（3–4 题，重点）

**识别**："以下哪项最能加强 / 削弱上述论证"。

#### 4.3.1 论证结构

```
论据（前提）→ 论证（推理）→ 论点（结论）
```

#### 4.3.2 加强方法（按力度）

| 方法 | 说明 | 力度 |
|------|------|------|
| 补充论据 | 直接支持论点 | 最强 |
| 建立联系 | 论据与论点之间搭桥 | 强 |
| 排除他因 | 排除其他可能 | 中 |
| 举例支持 | 举符合论点的例子 | 弱 |

#### 4.3.3 削弱方法（按力度）

| 方法 | 说明 | 力度 |
|------|------|------|
| 否定论点 | 直接否定结论 | 最强 |
| 断开联系 | 论据推不出论点 | 强 |
| 因果倒置 | 把因果颠倒 | 强 |
| 另有他因 | 提出其他原因 | 中 |
| 举例反驳 | 举反例 | 弱 |

#### 4.3.4 加强削弱力度比较

**削弱**：否定论点 > 断开联系 ≈ 因果倒置 > 另有他因 > 举例反驳

**加强**：补充论据 > 建立联系 > 排除他因 > 举例支持

### 4.4 前提假设（1–2 题）

**识别**："以下哪项是上述论证的前提 / 假设"。

**方法**：搭桥法。
- 找论据和论点中的不同概念
- 建立两者的联系
- 这个联系就是前提

### 4.5 解释评价（1 题）

**识别**："以下哪项最能解释上述现象 / 矛盾"。

**方法**：找能解释矛盾两面的选项。

### 4.6 归纳推导（1 题）

**识别**："由此可以推出"。

**方法**：
- 答案必须能从题干推出
- 不能过度推断
- 注意"可能""有些"等弱化词

## 七、真题示例

### 示例 1（图形—数量规律）

**题目**：给出一组图形（5 个），均为封闭图形，问第 6 个应是什么？

**解析**：
- 元素不同 → 看数量
- 数封闭区域：3, 4, 5, 6, 7 → 等差数列
- 第 6 个应为 8 个封闭区域
- 选有 8 个封闭区域的选项

---

### 示例 2（图形—一笔画）

**题目**：以下哪个图形可一笔画成？

**解析**：
- 数奇点（引出奇数条线的点）
- 奇点数 = 0 或 2 → 可一笔画
- 逐个选项数奇点
- 选奇点数为 0 或 2 的

---

### 示例 3（定义判断）

**定义**：锚定效应是指人们在做决策时，过度依赖最初获得的信息（锚），并以此为基准调整后续判断的现象。

**属于锚定效应的是**：
A. 小明看到商品原价 999 元，现价 499 元，觉得很划算
B. 小红比较了多家店铺后选择最便宜的
C. 小李根据朋友推荐购买商品
D. 小张凭直觉做决定

**解析**：
- 要素：①做决策时 ②依赖最初信息（锚）③以此为基准调整
- A：看到原价 999（锚）→ 觉得 499 划算 → 符合
- B：比较多家，无"最初信息"锚定 → 不符合
- C：朋友推荐，非"最初信息"锚定 → 不符合
- D：凭直觉，无锚 → 不符合

**答案**：A

---

### 示例 4（类比推理）

**题目**：医生：医院：治病
A. 教师：学校：教书  B. 工人：工厂：生产  C. 农民：土地：种地  D. 商人：商店：卖货

**解析**：
- 题干：医生在医院治病（职业：场所：行为）
- A 教师：学校：教书 → 符合
- B 工人：工厂：生产 → 符合
- C 农民：土地：种地 → 土地不是机构
- D 商人：商店：卖货 → 符合
- 二级辨析：医生、教师都是专业技术人员，场所都是事业单位
- 选 A

**答案**：A

### 示例 5（翻译推理）

**题目**：如果天下雨，那么地会湿。以下哪项一定正确？
A. 地湿了，说明天下雨了
B. 天没下雨，地一定没湿
C. 地没湿，说明天没下雨
D. 天下雨了，地一定湿

**解析**：
- 翻译：下雨 → 地湿
- A 肯后，错
- B 否前，错
- C 逆否（¬地湿 → ¬下雨），对
- D 肯前，对

**答案**：D（注意"一定正确"，D 是直接推理，C 是逆否等价，都对，但 D 更直接）

### 示例 6（加强削弱）

**论证**：研究发现，每天喝咖啡的人患心脏病的概率低于不喝咖啡的人。因此，喝咖啡能预防心脏病。

**最能削弱的是**：
A. 咖啡含有咖啡因，可能影响睡眠
B. 每天喝咖啡的人通常也更注重锻炼和饮食
C. 有些喝咖啡的人也患心脏病
D. 咖啡的价格较高

**解析**：
- 论点：喝咖啡能预防心脏病
- 论据：喝咖啡的人患病率低
- 削弱方法：另有他因
- B：指出喝咖啡的人本身更健康（锻炼饮食），是这些因素导致患病率低，不是咖啡 → 另有他因，削弱
- A、D 无关
- C 举例反驳，力度弱

**答案**：B

## 八、配套速记口诀

- **图形规律**：相同看位置，相似看样式，不同看数量，都不行看属性
- **一笔画**：奇点零或二，才能一笔画
- **定义要素**：主体客体目的方式，逐一比对
- **类比方向**：A 到 B 还是 B 到 A，方向不能反
- **包含组成**：包含是一种，组成是一部分
- **翻译推理**：前推后，后推前，逆否等价才安全
- **加强削弱**：否论点最强，举例最弱，另有他因中等
