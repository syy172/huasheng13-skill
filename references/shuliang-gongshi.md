# 数量关系公式速查手册（花生十三体系）

本文件是 `shuliang-guanxi.md` 的补充，按题型分类汇总所有核心公式、速解技巧，配记忆口诀，是数量关系的"公式字典"。

## 一、工程问题公式

### 1.1 基础公式

| 公式 | 说明 |
|------|------|
| 工作总量 = 效率 × 时间 | 核心公式 |
| 合作效率 = 各效率之和 | 合作问题 |
| 合作时间 = 总量 / 合作效率 | 合作问题 |

### 1.2 赋值法

| 情况 | 赋值方法 |
|------|---------|
| 给定时间 | 赋总量 = 时间最小公倍数 |
| 给定效率比 | 赋效率 = 比例值 |
| 给定人数/机器 | 赋每人/每台效率 = 1 |

### 1.3 变体公式

| 公式 | 说明 |
|------|------|
| 新效率 = 原效率 × (1 + 提升比例) | 效率提升 |
| 新时间 = 总量 / 新效率 | 效率变化后时间 |
| 实际工作 = 效率 × 实际时间 | 中途休息 |

## 二、行程问题公式

### 2.1 基础公式

| 公式 | 说明 |
|------|------|
| 路程 = 速度 × 时间 | 核心公式 |
| 速度 = 路程 / 时间 | 求速度 |
| 时间 = 路程 / 速度 | 求时间 |

### 2.2 相遇追及

| 公式 | 说明 |
|------|------|
| 相遇路程 = 速度和 × 相遇时间 | 相遇问题 |
| 追及路程 = 速度差 × 追及时间 | 追及问题 |
| 相遇时间 = 路程和 / 速度和 | 求相遇时间 |
| 追及时间 = 路程差 / 速度差 | 求追及时间 |

### 2.3 流水行船

| 公式 | 说明 |
|------|------|
| 顺水速度 = 船速 + 水速 | 顺水 |
| 逆水速度 = 船速 - 水速 | 逆水 |
| 船速 = (顺水 + 逆水) / 2 | 求船速 |
| 水速 = (顺水 - 逆水) / 2 | 求水速 |

### 2.4 电梯问题

| 公式 | 说明 |
|------|------|
| 顺行：人速 + 梯速 | 顺电梯 |
| 逆行：人速 - 梯速 | 逆电梯 |

### 2.5 环形问题

| 公式 | 说明 |
|------|------|
| 同向相遇：路程差 = 周长 × n | 同向追及 |
| 相向相遇：路程和 = 周长 × n | 相向相遇 |

### 2.6 单位换算

| 换算 | 说明 |
|------|------|
| 1 m/s = 3.6 km/h | 速度换算 |
| 1 km = 1000 m | 长度换算 |
| 1 小时 = 60 分 = 3600 秒 | 时间换算 |

## 三、利润问题公式

### 3.1 基础公式

| 公式 | 说明 |
|------|------|
| 利润 = 售价 - 成本 | 利润定义 |
| 利润率 = 利润 / 成本 × 100% | 利润率（分母是成本） |
| 售价 = 成本 × (1 + 利润率) | 求售价 |
| 成本 = 售价 / (1 + 利润率) | 求成本 |
| 折扣价 = 原价 × 折扣 | 打折 |

### 3.2 赋值法

| 情况 | 赋值方法 |
|------|---------|
| 涉及利润率 | 赋成本 = 100 |
| 涉及折扣 | 赋原价 = 100 |

### 3.3 变体公式

| 公式 | 说明 |
|------|------|
| 总利润 = 单件利润 × 销量 | 总利润 |
| 新售价 = 原售价 × 折扣 | 打折后 |
| 新利润率 = (新售价 - 成本) / 成本 | 打折后利润率 |

## 四、容斥问题公式

### 4.1 两集合

| 公式 | 说明 |
|------|------|
| A + B - AB = 总 - 都不 | 两集合容斥 |
| AB = A + B - (总 - 都不) | 求交集 |
| 都不 = 总 - A - B + AB | 求都不 |

### 4.2 三集合（标准型）

| 公式 | 说明 |
|------|------|
| A + B + C - AB - AC - BC + ABC = 总 - 都不 | 三集合标准 |

### 4.3 三集合（非标准型）

| 公式 | 说明 |
|------|------|
| A + B + C - 只满足两个 - 2×满足三个 = 总 - 都不 | 三集合非标准 |

### 4.4 容斥图解

```
        A
       / \
      /   \
     AB---AC
      \   /
       \ /
        ABC
       / \
      /   \
     BC----
      \   /
       \ /
        B---C
```

## 五、排列组合公式

### 5.1 基础公式

| 公式 | 说明 |
|------|------|
| A(n,m) = n! / (n-m)! | 排列（有序） |
| C(n,m) = n! / [m! × (n-m)!] | 组合（无序） |
| C(n,m) = C(n,n-m) | 组合性质 |
| C(n,m) = C(n-1,m-1) + C(n-1,m) | 杨辉三角 |

### 5.2 常用组合数

| C(n,m) | 值 |
|--------|---|
| C(2,1) | 2 |
| C(3,1) | 3 |
| C(3,2) | 3 |
| C(4,2) | 6 |
| C(5,2) | 10 |
| C(6,2) | 15 |
| C(5,3) | 10 |
| C(6,3) | 20 |

### 5.3 解题方法

| 方法 | 适用 |
|------|------|
| 捆绑法 | 相邻问题 |
| 插空法 | 不相邻问题 |
| 隔板法 | 相同物品分配 |
| 间接法 | 正面复杂时 |
| 分类法 | 情况较多时 |

### 5.4 捆绑法步骤

1. 把要相邻的元素捆绑成整体
2. 把整体和其他元素全排列
3. 整体内部全排列
4. 相乘得结果

### 5.5 插空法步骤

1. 先排其他元素
2. 形成空位（n 个元素有 n+1 个空）
3. 把不相邻元素插入空位
4. 相乘得结果

## 六、概率问题公式

### 6.1 基础公式

| 公式 | 说明 |
|------|------|
| P(A) = A 的情况数 / 总情况数 | 古典概型 |
| P(A) = A 的长度 / 总长度 | 几何概型 |
| P(A∪B) = P(A) + P(B) - P(A∩B) | 加法公式 |
| P(A∩B) = P(A) × P(B) | 乘法公式（独立） |
| P(¬A) = 1 - P(A) | 对立事件 |

### 6.2 常见概率

| 情况 | 公式 |
|------|------|
| 至少一个 | 1 - 都不 |
| 都发生 | 各概率相乘 |
| 都不发生 | 各不概率相乘 |
| 恰好 k 个 | C(n,k) × p^k × (1-p)^(n-k) |

## 九、溶液问题公式

### 9.1 基础公式

| 公式 | 说明 |
|------|------|
| 浓度 = 溶质 / 溶液 × 100% | 浓度定义 |
| 溶液 = 溶质 + 溶剂 | 溶液组成 |
| 溶质 = 溶液 × 浓度 | 求溶质 |

### 9.2 混合溶液

| 公式 | 说明 |
|------|------|
| 混合浓度 = 总溶质 / 总溶液 | 混合 |
| 十字交叉法 | 两种溶液混合 |

### 9.3 十字交叉法

```
溶液A浓度    混合浓度-溶液B浓度
        \   /
         混合浓度
        /   \
溶液B浓度    溶液A浓度-混合浓度
```

**示例**：20% 和 30% 混合成 25%
- 20% ... 5%（30-25）
- 30% ... 5%（25-20）
- 比例 1:1

### 9.4 蒸发稀释

| 操作 | 变化 |
|------|------|
| 蒸发 | 溶剂减少，溶质不变 |
| 稀释 | 溶剂增加，溶质不变 |
| 加溶质 | 溶质增加，溶剂不变 |

## 十、牛吃草问题公式

### 10.1 核心公式

| 公式 | 说明 |
|------|------|
| 草长速度 = (N1×T1 - N2×T2) / (T1 - T2) | 草长速度 |
| 原草量 = N1×T1 - 草长速度×T1 | 原有草量 |
| 吃草天数 = 原草量 / (N - 草长速度) | 求天数 |
| 牛数 = 原草量 / 天数 + 草长速度 | 求牛数 |

### 10.2 适用场景

- 牛吃草
- 水池进水排水
- 收银台排队
- 资源开采

## 十五、公式速记口诀

### 15.1 工程问题
- 时间给最小公倍数，效率给比例
- 合作效率相加，合作时间相除

### 15.2 行程问题
- 相遇速度和，追及速度差
- 顺水加逆水减，船速中水速差

### 15.3 利润问题
- 成本设一百，利润率好算
- 利润率分母是成本，不是售价

### 15.4 容斥问题
- 两集合：A 加 B 减交集
- 三集合：加三减二加一

### 15.5 排列组合
- 有序排列，无序组合
- 相邻捆绑，不相邻插空

### 15.6 概率问题
- 至少一个用补集，都发生用乘法
- 古典概型除法，几何概型长度比

### 15.7 年龄问题
- 年龄差不变，倍数随时间变
- 过去减年数，未来加年数

### 15.8 几何问题
- 周长面积要分清，体积立方要记牢
- 勾股定理三 四 五，相似三角形对应边

### 15.9 溶液问题
- 浓度溶质比溶液，混合用十字交叉
- 蒸发溶剂减，稀释溶剂加

### 15.10 牛吃草
- 先算草长速度，再算原有草量
- 天数等于原草除以净消耗

### 15.11 鸡兔同笼
- 假设全鸡，差除二得兔
- 通用公式：差除差得高分

### 15.12 最值问题
- 最不利加一才保证
- 抽屉原理 n 乘 k 减一加一
