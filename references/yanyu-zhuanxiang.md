# 言语理解专项突破（花生十三体系）

本文件是 `yanyu-lianjie.md` 和 `yanyu-ciyu.md` 的补充，针对言语理解各子题型提供专项突破方法。

---

## 一、逻辑填空专项

### 1.1 一空题

**解题流程**：
1. 读空格前后，找对应关系
2. 判断对应类型（转折/并列/递进/解释）
3. 推导空格含义
4. 选最匹配的词

**示例**：
- 题目：这项政策______，解决了群众反映强烈的问题。
- 选项：A. 立竿见影  B. 行之有效  C. 卓有成效  D. 事半功倍
- 语境：政策有效，解决了问题
- A 立竿见影：见效快
- B 行之有效：实行起来有效
- C 卓有成效：成绩显著
- D 事半功倍：费力小收效大
- "解决了问题"强调成效，选 C

### 1.2 两空题

**解题流程**：
1. 先看第一空，排除明显错的
2. 再看第二空，验证剩余选项
3. 两空都符合的为答案

**示例**：
- 题目：面对复杂局面，我们需要______，不能______。
- 选项：A. 标本兼治 / 扬汤止沸  B. 釜底抽薪 / 饮鸩止渴  C. 对症下药 / 因地制宜  D. 有的放矢 / 无的放矢
- 第一空：应对复杂局面需要"治本"
  - A 标本兼治：治标又治本 ✓
  - B 釜底抽薪：根本解决 ✓
  - C 对症下药：针对问题 ✓
  - D 有的放矢：有目标 ✓
- 第二空：不能怎样（反义）
  - A 扬汤止沸：治标不治本 ✓（反义）
  - B 饮鸩止渴：短视 ✓（反义，但程度重）
  - C 因地制宜：不是反义 ✗
  - D 无的放矢：无目标 ✓（反义）
- A 最匹配，选 A

### 1.3 成语辨析题

**解题要点**：
1. 理解成语本义
2. 注意感情色彩
3. 注意程度轻重
4. 注意适用对象

**常见陷阱**：
- 望文生义：只看字面
- 褒贬误用：褒义贬用或反之
- 程度不当：轻重不匹配
- 对象不当：用错对象

## 二、片段阅读专项

### 2.1 中心理解题

**解题流程**：
1. 找关联词（转折/递进/因果/条件/并列）
2. 定位重点句
3. 找主题词
4. 选含主题词的重点句同义替换

**关联词优先级**：
1. 转折（但是、然而）→ 转折后是重点
2. 递进（而且、甚至）→ 递进后是重点
3. 因果（因此、所以）→ 结果是重点
4. 条件（只有、只要）→ 条件是重点
5. 并列（同时、此外）→ 全面概括

**主题词识别**：
- 反复出现的词
- 关联词附近的词
- 首尾句的词

**选项分析**：
- 含主题词 → 保留
- 不含主题词 → 排除
- 主题词范围扩大/缩小 → 排除
- 非重点句内容 → 排除

### 2.2 细节判断题

**解题流程**：
1. 看选项，定位原文
2. 逐项比对
3. 找偷换/无中生有/绝对化

**六种偷换**：
| 类型 | 说明 | 示例 |
|------|------|------|
| 概念偷换 | 换概念 | "城市"→"农村" |
| 时态偷换 | 换时态 | "将"→"已" |
| 范围偷换 | 换范围 | "部分"→"全部" |
| 语气偷换 | 换语气 | "可能"→"一定" |
| 有无偷换 | 换有无 | "有"→"无" |
| 正反偷换 | 换正反 | "是"→"不是" |

**绝对化选项**：
- "一定""必须""所有""都""完全"→ 多为错
- "可能""也许""一些""部分"→ 多为对

### 2.3 标题填入题

**解题流程**：
1. 找中心句
2. 提炼主题
3. 选吸引人且准确的标题

**好标题标准**：
- 准确：反映主旨
- 简洁：字数适中
- 吸引：有吸引力
- 全面：涵盖要点

### 2.4 语句衔接题

**解题流程**：
1. 看空格位置（首/中/尾）
2. 找前后话题
3. 选话题一致的

**位置规律**：
- 首句：引出话题
- 中句：承接上文，启下文
- 尾句：总结全文

### 2.5 语句排序题

**解题流程**：
1. 看选项，确定首句
2. 找捆绑（代词、关联词）
3. 确定顺序

**首句判断**：
- 代词（这、那、其）不做首句
- 关联词后半（但是、所以）不做首句
- 总结性句子不做首句
- 引出话题的适合做首句

**捆绑判断**：
- 代词 + 指代对象
- 关联词前后
- 时间顺序
- 逻辑顺序

## 三、语句表达专项

### 3.1 语句排序

**方法**：
1. 看选项确定首句（排除代词、关联词后半）
2. 找捆绑（代词指代、关联词、时间、逻辑）
3. 确定尾句（总结性）
4. 验证顺序

### 3.2 语句填入

**方法**：
1. 看位置（首/中/尾）
2. 看前后话题
3. 选话题一致、逻辑通顺的

### 3.3 下文推断

**方法**：
1. 看尾句话题
2. 选与尾句话题一致的

## 四、篇章阅读专项

### 4.1 篇章阅读特点

- 题量大（一篇文章 5 题）
- 阅读量大
- 时间紧
- 考查综合能力

### 4.2 解题策略

**先看题后读文**：
1. 先看题目，了解考查点
2. 带着问题读文章
3. 定位相关段落
4. 作答

**题型分类**：
- 中心理解：找主旨
- 细节判断：比对
- 词句理解：联系上下文
- 标题填入：概括
- 语句填入：衔接

### 4.3 时间分配

- 阅读文章：3-4 分钟
- 作答 5 题：5-6 分钟
- 总用时：8-10 分钟

## 五、言语理解提速技巧

### 5.1 阅读提速

**技巧**：
1. 跳读：跳过例子、数据
2. 扫读：快速找关键词
3. 精读：重点句精读
4. 结构读：抓文章结构

### 5.2 答题提速

**技巧**：
1. 先看问题再读文
2. 带着问题找答案
3. 选项对比排除
4. 不纠结，快速选

### 5.3 训练方法

**第一阶段**：精做（不限时，求对）
**第二阶段**：限时（求快求对）
**第三阶段**：套题（模拟考场）

## 六、配套速记口诀

- **填空**：一空找对应，两空先排后验
- **成语**：本义色彩程度对象
- **中心**：关联词定重点，主题词必含
- **细节**：六种偷换要防，绝对化多为错
- **排序**：代词不做首，捆绑定顺序
- **篇章**：先看题后读文，定位找答案
- **提速**：跳读扫读精读，先问题后文章
