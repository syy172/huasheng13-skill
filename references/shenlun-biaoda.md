# 申论规范表达与金句积累（花生十三体系）

本文件是 `shenlun.md` 的补充，收录申论规范表达、政府工作用语、金句素材，是申论写作的"语言库"。

## 一、申论规范表达原则

### 1.1 规范化要求

| 要求 | 说明 | 示例 |
|------|------|------|
| 书面化 | 不用口语 | "搞"→"开展"、"弄"→"推进" |
| 规范化 | 用政府用语 | "想办法"→"研究对策" |
| 简洁化 | 不啰嗦 | "进行了研究讨论"→"研究讨论" |
| 逻辑化 | 有条理 | 用"一是、二是、三是" |
| 客观化 | 不主观 | "我觉得"→"据了解" |

### 1.2 口语转规范表达

| 口语 | 规范表达 |
|------|---------|
| 搞 | 开展、推进、实施 |
| 弄 | 推进、落实 |
| 弄好 | 落实到位、取得成效 |
| 想办法 | 研究对策、制定措施 |
| 出问题 | 出现问题、存在隐患 |
| 解决了 | 有效解决、妥善处理 |
| 做得好 | 成效显著、扎实推进 |
| 不够好 | 有待加强、仍需完善 |
| 很多 | 较多、众多、大量 |
| 很少 | 较少、稀缺、匮乏 |
| 越来越多 | 日益增多、持续增加 |
| 越来越好 | 持续改善、稳步提升 |
| 大家都 | 各界、各方、普遍 |
| 一起 | 共同、协同、合力 |
| 因为 | 由于、源于、基于 |
| 所以 | 因此、故、由此可见 |
| 但是 | 然而、但、不过 |
| 如果 | 若、倘若、假使 |

## 三、申论金句积累

### 3.1 开头金句

#### 3.1.1 发展类

- "发展是硬道理，是解决一切问题的基础和关键。"
- "高质量发展是全面建设社会主义现代化国家的首要任务。"
- "创新是引领发展的第一动力，协调是持续健康发展的内在要求。"
- "绿水青山就是金山银山，保护生态环境就是保护生产力。"

#### 3.1.2 民生类

- "人民对美好生活的向往，就是我们的奋斗目标。"
- "民生无小事，枝叶总关情。"
- "利民之事，丝发必兴；厉民之事，毫末必去。"
- "民之所忧，我必念之；民之所盼，我必行之。"

#### 3.1.3 治理类

- "治国安邦，重在基层；管党治党，重在基础。"
- "郡县治，天下安；乡村治，百姓安。"
- "基层强则国家强，基层安则天下安。"
- "治理有效是乡村振兴的重要保障。"

#### 3.1.4 文化类

- "文化是一个国家、一个民族的灵魂。"
- "文化兴国运兴，文化强民族强。"
- "没有高度的文化自信，没有文化的繁荣兴盛，就没有中华民族伟大复兴。"
- "中华优秀传统文化是中华民族的根和魂。"

### 3.2 过渡金句

#### 3.2.1 转折过渡

- "然而，我们也要清醒地看到，......"
- "在肯定成绩的同时，也要正视存在的问题。"
- "尽管取得了显著成效，但......仍任重道远。"
- "成绩值得肯定，问题不容忽视。"

#### 3.2.2 递进过渡

- "不仅如此，......更是......"
- "更重要的是，......"
- "在此基础上，......"
- "进一步说，......"

#### 3.2.3 因果过渡

- "之所以......，是因为......"
- "......的根本原因在于......"
- "正是基于此，......"
- "由此可见，......"

### 3.3 结尾金句

#### 3.3.1 号召类

- "让我们携手同心，为实现......而努力奋斗！"
- "征程万里风正劲，重任千钧再出发。"
- "路虽远，行则将至；事虽难，做则必成。"
- "行而不辍，未来可期。"

#### 3.3.2 展望类

- "展望未来，......必将......"
- "站在新的历史起点上，......"
- "面向未来，我们要......"
- "新时代新征程，......"

#### 3.3.3 总结类

- "总之，......"
- "综上所述，......"
- "由此可见，......"
- "实践证明，......"

### 3.4 分论点金句模板

#### 3.4.1 "是"字句

- "......是......的关键 / 核心 / 根本 / 前提 / 基础 / 保障"
- "......是......的必然要求 / 内在需要 / 重要举措"
- "......是......的必由之路 / 重要途径"

#### 3.4.2 "要"字句

- "要......，必须......"
- "要......，需要......"
- "要......，离不开......"

#### 3.4.3 "只有"句

- "只有......，才能......"
- "唯有......，方能......"
- "......方能......"

## 五、申论常见错误

### 5.1 内容错误

| 错误 | 说明 | 改进 |
|------|------|------|
| 脱离材料 | 答案不在材料中 | 紧扣材料 |
| 要点不全 | 漏掉关键点 | 多角度找点 |
| 不分类 | 要点堆砌 | 同类合并 |
| 照抄原文 | 不概括 | 提炼关键词 |
| 主观发挥 | 加自己观点 | 客观陈述 |

### 5.2 形式错误

| 错误 | 说明 | 改进 |
|------|------|------|
| 无序号 | 不标序号 | 用①②③ |
| 超字数 | 超过限制 | 精简表达 |
| 字迹乱 | 卷面不整 | 工整书写 |
| 格式错 | 公文格式不对 | 记住格式 |
| 涂改多 | 卷面不洁 | 想好再写 |

### 5.3 语言错误

| 错误 | 说明 | 改进 |
|------|------|------|
| 口语化 | 用口语 | 用书面语 |

## 六、申论写作练习

### 练习 1：单一题规范表达

**原文**："我们村的路太烂了，下雨天根本没法走，农产品运不出去，烂在地里。"

**规范表达**：
- 基础设施落后。道路状况差，影响农产品运输，造成经济损失。

### 练习 2：对策题规范表达

**原文**："政府要多给钱，多派人，多搞培训。"

**规范表达**：
- 一是加大资金投入。增加财政支持，保障工作经费。
- 二是加强人才配备。充实工作人员，优化队伍结构。
- 三是强化培训教育。开展专业培训，提升能力素质。

### 练习 3：大作文分论点

**主题**：乡村振兴

**分论点规范表达**：
- 产业振兴是乡村振兴的基础。要发展特色产业，延长产业链条，提升附加值。
- 人才振兴是乡村振兴的关键。要培育本土人才，吸引外来人才，留住青年人才。
- 文化振兴是乡村振兴的灵魂。要传承乡土文化，培育文明乡风，丰富精神生活。
- 生态振兴是乡村振兴的底色。要保护生态环境，发展绿色产业，建设美丽乡村。
- 组织振兴是乡村振兴的保障。要加强基层党建，完善治理体系，提升治理能力。

## 七、配套速记口诀

- **规范表达**：书面化、规范化、简洁化、逻辑化、客观化
- **口语转规范**：搞→开展，弄→推进，想办法→研究对策
- **政府用语**：推进推动推行，加强强化健全，深化拓展拓宽，破解化解解决
- **金句积累**：开头发展民生治理文化，过渡转折递进因果，结尾号召展望总结
- **论证素材**：人物改革创新奉献文化，事件科技民生，政策战略
- **避免错误**：不脱离材料，不照抄原文，不主观发挥
