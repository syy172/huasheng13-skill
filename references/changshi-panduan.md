# 常识判断（花生十三体系）

常识判断是行测中"会就会，不会就不会"的模块，共 20 题（国考），涉及政治、经济、法律、历史、文化、地理、环境、自然、科技等。花生十三体系强调 **"抓重点 + 巧蒙 + 不纠结"**，因为常识范围太广，不可能全背，关键是抓高频考点 + 用排除法。

---

## 一、总体策略

### 1.1 常识判断特点

- **范围广**：上知天文下知地理
- **记忆量大**：需要长期积累
- **性价比低**：花大量时间背，考试未必考
- **不可预测**：每年考点变化大

### 1.2 应对策略

| 策略 | 说明 |
|------|------|
| 抓高频 | 重点背每年必考的（时政、法律、党史） |
| 巧排除 | 用排除法提高蒙题命中率 |
| 不纠结 | 不会的 30 秒选一个，不浪费时间 |
| 重积累 | 平时多看新闻、多读常识 |

### 1.3 时间分配

- 每题不超过 30 秒
- 20 题总用时 8–10 分钟
- 不会的立即蒙，不纠结

## 二、八大模块高频考点

### 2.1 政治常识（4–6 题）

#### 2.1.1 马克思主义基本原理

**高频考点**：
- 哲学基本问题：思维与存在的关系
- 唯物论：物质决定意识
- 辩证法：对立统一、质量互变、否定之否定
- 认识论：实践是认识的基础
- 唯物史观：社会存在决定社会意识

**常考名言与哲理对应**：
| 名言 | 哲理 |
|------|------|
| 水滴石穿 | 质量互变 |
| 刻舟求剑 | 静止看问题 |
| 守株待兔 | 把偶然当必然 |
| 掩耳盗铃 | 主观唯心 |
| 画蛇添足 | 过犹不及 |

#### 2.1.2 毛泽东思想

- 新民主主义革命理论
- 三大法宝：统一战线、武装斗争、党的建设
- 农村包围城市道路
- 实事求是、群众路线、独立自主（活的灵魂）

#### 2.1.3 中国特色社会主义理论

- 邓小平理论：社会主义市场经济、一国两制
- 三个代表：代表先进生产力、先进文化、人民利益
- 科学发展观：以人为本、全面协调可持续
- 习近平新时代中国特色社会主义思想：十四个坚持

#### 2.1.4 党史国史

**重要会议**：
| 会议 | 时间 | 意义 |
|------|------|------|
| 一大 | 1921 | 党的成立 |
| 遵义会议 | 1935 | 转折点 |
| 七大 | 1945 | 毛泽东思想确立 |
| 十一届三中全会 | 1978 | 改革开放 |
| 十九大 | 2017 | 新时代 |
| 二十大 | 2022 | 中国式现代化 |

### 2.2 经济常识（2–3 题）

#### 2.2.1 基本概念

| 概念 | 含义 |
|------|------|
| GDP | 国内生产总值（地域） |
| GNP | 国民生产总值（国民） |
| CPI | 消费者物价指数（通胀） |
| PPI | 生产者物价指数 |
| 恩格尔系数 | 食品支出占总支出（越低越富裕） |
| 基尼系数 | 贫富差距（0.4 警戒线） |

#### 2.2.2 宏观调控

**财政政策**：
- 扩张性（经济低迷）：增支减税
- 紧缩性（经济过热）：减支增税

**货币政策**：
- 扩张性：降息降准
- 紧缩性：加息加准

#### 2.2.3 市场经济

- 市场调节：价格、供求、竞争
- 市场失灵：垄断、外部性、公共品、信息不对称

### 2.3 法律常识（3–4 题）

#### 2.3.1 宪法

- 国家根本大法
-公民基本权利：平等权、选举权、言论自由、人身自由
- 国家机构：人大、政府、监委、法院、检察院

#### 2.3.2 民法典

- 总则、物权、合同、人格权、婚姻家庭、继承、侵权责任
- 民事行为能力：完全（18+）、限制（8-18）、无（<8）
- 诉讼时效：3 年

#### 2.3.3 刑法

- 犯罪构成：主体、客体、主观、客观
- 刑罚：主刑（管制、拘役、有期、无期、死刑）、附加刑
- 年龄：12、14、16 周岁节点

#### 2.3.4 行政法

- 行政处罚：警告、罚款、没收、责令停产、拘留
- 行政许可：普通许可、特许、认可、核准、登记
- 行政复议：60 日内
- 行政诉讼：6 个月内

## 三、蒙题技巧

### 3.1 排除法

**方法**：排除明显错误的，提高蒙题命中率。

**示例**：
- 题目问"以下哪个是正确的"
- A、B 明显错 → 在 C、D 中蒙

### 3.2 常识法

**方法**：用生活常识判断。

**示例**：
- "夏天穿浅色衣服更凉快"→ 对（浅色反光）
- "冬天穿深色衣服更暖和"→ 对（深色吸热）

### 3.3 绝对化排除

**方法**：含"一定""必须""所有""都"的选项多为错。

**示例**：
- "所有细菌都是有害的"→ 错（有有益菌）
- "必须...才能..."→ 多为错

### 3.4 选项分布

- 国考常识答案分布较均匀
- 已确定 5 题答案，若 A 出现 3 次，剩余题少蒙 A

## 六、真题示例

### 示例 1（政治常识）

**题目**：下列关于习近平新时代中国特色社会主义思想的说法，正确的是：
A. 是马克思主义中国化的最新成果
B. 是党和人民实践经验和集体智慧的结晶
C. 是中国特色社会主义理论体系的重要组成部分
D. 以上都对

**解析**：A、B、C 都正确，选 D。

**答案**：D

---

### 示例 2（法律常识）

**题目**：根据《民法典》，下列属于完全民事行为能力人的是：
A. 7 周岁的小明
B. 15 周岁的小红
C. 18 周岁的小李
D. 16 周岁的小张（以自己劳动收入为主要生活来源）

**解析**：
- 完全民事行为能力：18 周岁以上，或 16 周岁以上以劳动收入为主要生活来源
- C 和 D 都符合，但题目问"属于"，选最典型的

**答案**：C

---

### 示例 3（历史人文）

**题目**：下列历史事件按时间先后排序正确的是：
A. 秦统一六国—文景之治—贞观之治—靖康之变
B. 文景之治—秦统一六国—贞观之治—靖康之变
C. 秦统一六国—贞观之治—文景之治—靖康之变
D. 文景之治—贞观之治—秦统一六国—靖康之变

**解析**：
- 秦统一：前 221
- 文景之治：西汉（前 180-前 141）
- 贞观之治：唐（627-649）
- 靖康之变：北宋（1127）
- 顺序：秦→文景→贞观→靖康

**答案**：A

---

### 示例 4（地理常识）

**题目**：关于我国地理，下列说法正确的是：
A. 我国地势东高西低
B. 我国最大的湖泊是青海湖
C. 长江发源于唐古拉山
D. 我国最南端是曾母暗沙

**解析**：
- A 错，西高东低
- B 青海湖是最大咸水湖，最大湖泊也是青海湖（对的）
- C 长江发源于唐古拉山（对的）
- D 曾母暗沙（对的）
- B、C、D 都对，但题目可能问"最"，选 B

**答案**：B

---

B. 嫦娥工程是探月工程
C. 天问一号是火星探测器
D. 以上都对

**解析**：A、B、C 都正确。

**答案**：D
