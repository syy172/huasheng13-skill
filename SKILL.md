# 公考花生十三技巧 Skill

基于花生十三公开教学体系整理的公务员考试（国考 / 省考 / 联考 / 事业单位）行测 + 申论解题技巧库。覆盖六大模块、二十余题型、上百条速算与解题公式，配真题示例与实战复盘方法。

## 二、核心工作流

### Step 1：识别用户场景

| 用户输入              | 场景    | 进入路径                   |
| ----------------- | ----- | ---------------------- |
| "这道题怎么做" + 题目     | 单题讲解  | Step 2A                |
| "XX 模块怎么学 / 怎么提速" | 模块方法论 | Step 2B                |
| "帮我做一套题 / 套题复盘"   | 套题实战  | Step 2C                |
| "怎么安排备考 / 多久能上岸"  | 备考规划  | Step 2D                |
| "行测怎么分配时间 / 做题顺序" | 策略咨询  | references/strategy.md |

### Step 2A：单题讲解流程

1. **识别题型**：判断属于哪个模块哪个子题型（如"资料分析—增长率比较"）
2. **调用对应 references 文件**：读取该题型的标准解题步骤
3. **套用方法**：按"识别标志 → 套公式 / 套方法 → 速算 → 选答案"四步走
4. **给出解析**：包含「题型识别 → 思路 → 计算 → 答案 → 易错点 → 同类题拓展」
5. **附提速技巧**：如果是资料 / 数量，必须给出速算方法（截位直除 / 415 份数 / 假设分配等）

### Step 2B：模块方法论讲解

读取对应 references 文件，按"题型分类 → 核心方法 → 速算 / 速解技巧 → 高频考点 → 易错点 → 真题示例"输出。

### Step 2C：套题实战与复盘

1. 让用户上传 / 粘贴套题（或指定年份真题）
2. 按真实考试时间分配建议用户作答
3. 收集用户答案后，逐题讲解错题
4. 输出复盘报告：模块正确率、用时分布、薄弱题型、改进建议
5. 参考 `references/strategy.md` 的复盘模板

### Step 2D：备考规划

收集：考试类型（国 / 省）、目标岗位、当前水平（模考分数）、剩余时间、每天可投入时长。按"基础 → 强化 → 冲刺"三阶段输出计划，每阶段明确模块重点与刷题量。

## 四、答题顺序与时间分配（核心策略）

### 推荐顺序（花生十三公开建议版）

```
常识（8 min）→ 资料分析（28 min）→ 判断推理（35 min）→ 言语理解（35 min）→ 数量关系（剩余时间选做）
```

### 时间分配原则

1. **先易后难、先高分后低分**：资料分析分值密度最高，必须先做且做对
2. **数量关系放最后**：性价比最低，留 5–8 分钟挑 5–7 道会做的，其余统一蒙
3. **常识不纠结**：会就选，不会就蒙，单题不超过 30 秒
4. **涂卡分两次**：做完资料涂一次，做完言语判断涂一次，避免最后来不及

详细策略见 `references/strategy.md`。

## 六、输出规范

### 单题讲解输出格式

```
【题型】资料分析—增长率计算
【题干】（用户题目）
【题型识别】关键词"比去年同期增长 X%"→ 增长率计算题
【解题思路】
  1. 定位材料：找到现期量与基期量
  2. 套公式：增长率 =（现期 - 基期）/ 基期
  3. 速算：截位直除（保留前 3 位）
【计算过程】...
【答案】X
【易错点】① 时间定位错 ② 单位（万 / 亿）③ 增长量与增长率混淆
【同类拓展】若题目问"增长量"，改用 415 份数法
```

## 七、使用约束

1. **速算必须给过程**：资料分析、数量关系的题，不能只给答案，必须展示截位直除 / 415 份数等速算过程
2. **方法优先于计算**：先讲识别标志和解题思路，再讲计算
3. **易错点必给**：每道题、每个方法都要附易错点
4. **真题示例用通用数据**：示例题目使用公开真题或自编数据，不照搬任何付费课程原题
5. **申论大作文给框架**：不给完整范文，给"立意 + 结构 + 分论点 + 素材方向"，让用户自己写
6. **备考建议因人而异**：必须先问清用户水平、时间、目标，再给计划，不套模板
